VS Code とは

VS Code は、主にプログラミングやマークダウンなどを書くときに使うエディタです。
メモ帳でプログラミングするわけにはいきませんよね。

VS Code を使うことで開発の効率アップが期待できます。
VS Code には、メモ帳にはない(であろう)さまざまな機能があります。
軽く紹介すると、ざっと以下の様になります。
  • コードのハイライトやフォーマット
  • ファイルマネージャー/ターミナル/エディタ/ビューワーを 1 つのウィンドウで使える
  • デバッガなどの便利機能
  • 拡張機能
この記事では、これらを簡単に紹介していきます。
