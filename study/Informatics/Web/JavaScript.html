ブラウザ API

ブラウザ API(Application Interface) とは、JavaScript プログラムが Web ブラウザのデータにアクセスするためのインターフェースです。代表的なものには、HTML 文書のデータにアクセスする document や、ブラウザのデバッグコンソールにアクセスする console があります。
