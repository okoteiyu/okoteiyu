FFT の n 乗根の制約


そのような係数を α とすると、

1 : n ( = 2 ^ x ) 種類の元を持ち、α^i ( 0 <= i <= n-1 ) の値それぞれが異なる値をとる。
2 : α^n = 1
3 : Σ(i : 0 → n-1 ) (α^k)^i の取る値が、 k = 0 ならば n , k ≠ 0 ならば 0 となる

Z/pZ ( p で割った余りを考える空間 ) で考えることにします。

ここで、n = 2 ^ x として、n 乗根 : g を考えると、Z/pZ の　FFT の n 乗根の制約より、g ^ n = 1 である必要があります。
ここで、フェルマーの小定理より、原始根 q と素数 p に対して、
     q^( p-1 ) = 1 が成り立ちます。

よって、p = a×2^x + 1 ( aは2と互いに素 ) と表すことで
     q^( p-1 ) = q^( a × 2 ^ x ) = ( q^a ) ^ ( 2 ^ x ) = 1 と表すことができ、
n = 2 ^ x より、
    (q^a) ^ n = 1 です。
g = q ^ a とおくと、g^n = 1 となり、g が FFT の n 乗根の制約の一つを満たすことがわかります。
