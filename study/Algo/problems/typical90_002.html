典型練習問題まとめ


重複組み合わせ(典型90 - 15)

                        
F(x) : 1 ~ N の整数から、どの 2 つの差の絶対値も x 以上であるようにいくつか取り出す方法の総数
                        
各 1 ≦ x ≦ N について、F(x) mod 1000000007 を計算してください。
                        
・N ≦ 100000
  • \(f(a,x)\) := \(a\) 個数取り出して、どの二つの差の絶対値も \(x\) 以上になる取り出し方の方法
    • を考えます。
    これは、以下のように考えることができます。
    • 赤いボールが \(a\) 個、白いボールが \(N - a\) 個あるものとする
    • \(a\) 個の赤いボールを一列に並べる
    • 隣り合う赤いボール同士の隙間全てに、\(x-1\) 個ずつ白いボールを入れる
         → 白いボールは残り \(N-a-(a-1)(x-1)\) 個になる
         → 赤いボール同士の距離が \(x\) になった
    • 残った白いボールは、赤いボール同士の間の領域 + 両端 の \(a+1\) 箇所に散りばめることができる
         → 残りの区別しない白いボールを、\(a+1\) 種類の箱に入れる組み合わせである。
    • このように並べたボールの左から、\(1 , 2 , ... , N\) と番号をつけると、
      赤いボールにつけられた番号が、選ばれる整数に対応している。 

    int main(){
        int n;cin >> n;
        ll mod = 1000000007;
        ModComb M(1000000,mod);
        M.COMinit();
        FOR(k,1,n+1){
            long long res = 0;
            FOR(a , 1 , n+1){
                ll p = n-a-(a-1)*(k-1) + a;
                if(p < 0)break;
                res += M.COM(p,a);
            }
            cout << res%mod << endl;
        }
        return 0;


    交点のカウント(典型 90 - 17)

                            
    円上で時計回りに 1 , 2 , ... , N と等間隔に並んだ N 個の点と、それらを結ぶ線分が M 本ある。
                        
    端点以外で線分同士が交差する交点の個数を出力せよ。

    円のどこかをカットして円を直線に変形させることで、数直線上での M 本の線分の問題に帰着します。

    数直線上において、ある線分(区間) \(L\) と交差する線分は、片方の端点が \(L\) の区間内にあり、もう片方の端点が \(L\) の区間外にある必要があります。
    ただし区間内とは、開区間 (端点を含まない) です。

    ここで、線分の左側の端点が小さい方から線分を見ていくことにし、\(i\) 番目に見る線分を \(L_i\) とします。
    すると各 \(i\) について \(i-1\) 番目以前の線分の左側の端点は全て \(L_i\) の区間外となります。
    線分が交差する条件より、 \(i-1\)番目以前の線分のうち、右端の端点が \(L_i\) の区間内にあるものは、\(L_i\) と交差することがわかります。

    既に調べた線分の右端の端点の位置を Set や セグメント木 に記録することで、\(L_i\) の区間内にあるものの個数をカウントできます。


    \(8\) が区間内なので答えに \(+1\)

    区間内に端点は存在しない

    \(8\) が区間内なので答えに \(+1\)

    まとめ


    これまでに見た線分の左端が必ず \(L_i\) の区間外であることを利用して、交点の個数を \(L_i\) の区間内にある右側端点の個数に帰着した。
    左端が \(L_i\) の区間内で、右端が区間外のパターンを見落としているように見えるが、そのようなパターンがあるなら、\(i+1\) 番目以降の線分を調べる時に \(L_i\) の右側の端点がカウントされることになるので問題ない。

    線分の端点のうち、点の番号が小さい方を左側の端点、大きい方を右側の端点とすると、円上でも同じ方法で交点を数えることができる。 
    int main(){
        ll n , m;cin >> n >> m;
        SplayTree<int> S;
        REP(i,n+1)S.push_back(0);
        vector<vll> segment(n+1);// [x] := 点 x に左端点がある線分の右端の位置の一覧
        REP(i,m){
            int left , right;cin >> left >> right;
            if(left>right)swap(left,right);
            segment[left].push_back(right); // 線分を 1 始まりの数直線上の区間と捉える
        }
        ll ans = 0;
        
        // 円を点 1 から始まる数直線と捉える
        FOR(left,1,n+1){
            for(int right : segment[left])if(left + 1 < right)ans += S.RangeSumQuery(left+1,right);// 区間内の右端の個数をカウント
            for(int right : segment[left])S.update_val(right,S[right].Value+1);// 右端をメモ
        }
        cout << ans << endl;
        return 0;


    区間DP (典型 90 - 19)

                            
    長さ N の 数列 A に対して、A が空になるまで以下の操作を繰り返す。
                        
    ・A の要素のうち、連続する 2 つの要素を選び、A から削除する。削除された要素以外の並び順は変わらない。
                        
    この操作におけるコストを、選んだ要素の値の差の絶対値とする。
                        
    各操作で好きな隣接要素を選べる時、操作を完了するためのコストの総和としての最小値を答えでください。
    \(A\) のどの区間に注目しても、考える問題は同じです。
    ある区間に対して操作を完了するためのコストに注目すると、その結果を用いて \(A\) 全体に対する操作完了コストの最小値を計算できる。
