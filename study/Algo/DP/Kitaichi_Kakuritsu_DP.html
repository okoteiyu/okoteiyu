確率的 Nim 

                        
初め、N 枚のカードがある。
                        
プレイヤーは以下の操作を繰り返す。
                        
・現在のカードの枚数を M とした時、0 ~ M が等確率で出るサイコロを振って出た目の枚数分だけカードを燃やす。
                        
・ただし、カードの枚数がちょうど 1 枚の際は、カードを燃やすことができず、ゲームオーバーとなる。
                        
プレイヤーが全てのカードを燃やし尽くす確率を計算してください。
・\(DP_X\) := 残りカード数が \(X\) 枚の状態から操作を始める場合の確率
とし、\(X\) が小さい方から DP テーブルを埋めていきます。
DP の式は、以下のようになります。
  • \(DP_X = \frac{DP_{X}}{X+1} + \frac{DP_{X-1}}{X+1} + \dots + \frac{DP_{1}}{X+1} + \frac{DP_{0}}{X+1}\)
サイコロは 0 の目も出るので、状態の遷移が起こらない場合も考慮しなければなりません。
その結果、左辺にも右辺にも \(DP_X\) が登場するので、\(DP_X\) の項を左辺に移動します。
\(DP_X\) は以下の式で求めることができます。
・\(DP_X = \sum_{1 \leq i \leq X}{\frac{DP_{X-i}}{X}} \)

確率的 Nim の期待値

                        
初め、N 枚のカードがある。
                        
プレイヤーは以下の操作を繰り返す。
                        
・現在のカードの枚数を M とした時、0 ~ M が等確率で出るサイコロを振って出た目の枚数分だけカードを燃やす。
                        
プレイヤーが全てのカードを燃やし尽くすまでにサイコロを振る回数の期待値を計算してください。
・\(DP_X\) := 残りカード数が \(X\) 枚の状態から操作を始める場合の期待値
とし、\(X\) が小さい方から DP テーブルを埋めていきます。
DP の式は、以下のようになります。
  • \(DP_X = 1 + \frac{DP_{X}}{X+1} + \frac{DP_{X-1}}{X+1} + \dots + \frac{DP_{1}}{X+1} + \frac{DP_{0}}{X+1}\)
サイコロを振る分 1 を加算し、出た目に応じた遷移先の状態からの期待値の平均を加算しています。
また、この式を代数的に解くことで以下の式を得られます。
  • \(DP_X = \frac{X+1}{X} + \sum_{1 \leq i \leq X}\frac{DP_{X-i}}{X}\)
これは、期待値 DP テンプレの記事で紹介したテンプレの式に当てはまります。
