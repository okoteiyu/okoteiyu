DPの典型として、期待値DPというものがあります。

期待値DPとは、ある状態 G から始まる(もしくは、G に到達するまでの)操作を終えるまでの操作回数の期待値を、DPを用いて求める手法です。

期待値とは


期待値を競プロっぽくいうと、運ゲーを攻略した時の結果の平均である。

ex.1 : 確率 \(0.1\) で \(5000\) 円当たり、確率 \(0.9\) で何ももらえない運ゲーの期待値は \(5000 \cdot 0.1 + 0\cdot 0.9 = 500円\) である。

ex.2 : 毎日一度サイコロを振り、1 以外が出たら帰宅できる職場で、最初に帰宅できるのは(期待値を考えると) 1.2 日目である。

ex1 による期待値の計算は、期待値の計算そのもの...
ex1 を言い換えると、\(5000\) 円札が \(1\) 枚と、\(0\) 円札が \(9\) 枚あり、これらの平均を求める問題と同じ。
逆に、平均値というのは各状態に重みをつけて和を求めたものでありますが、この重みというのは確率のことです。


ex2 は、ある事象 \(A\) が起こるまでの試行回数の期待値です。
この期待値は、\(A\) が確率 \(P\) で起こるとき \(\frac{1}{P}\) になると知られています。
こちらは頻出です。

期待値DPを説明する上で、もう一つ知っておかなければならないことがあります。



条件付き確率


事象Aが起きたとわかっている時、事象Bが起きる条件付き確率を \(P( B|A )\) とかきます。

ex. : サイコロを振った時、偶数が出たことがわかっているとします。その目が4以上である確率を求めてください。

直感的な回答をします。偶数は \(3\) つあるので分母が \(3\)。\(4\) 以上の値は \(2\) つあるので分子は \(2\)。よって\(\frac{2}{3}\) である。
ここで、分母に注目です。サイコロを振った場合の事象は6パターンあるので、本来サイコロを振る確率を考えるときは、分母は6のはずですが、今回は「偶数が出たとわかっている」という条件があるので、その条件に当てはまる事象の集合を分母にしました。

ここで、直感的な理解を手助けする考え方を見てみましょう(競プロではこれに近い考えで解きますので)。



最後に、\(P(B|A)\) の公式です。
\(P(s)\) を事象 \(s\) が起こる確率とします。
この時、\(P(B|A) = \frac{P(A\cap B) }{ P(A)} \) です。

さて、ここから本題になります。
\(DP_X :=\) 状態 \(X\) から、目的の状態に達するまでに行う試行回数の期待値 とします。

\(DP_X\) を \(X\) からの意味のある 遷移先の集合 \(S\) を用いて求めます。
ここで意味のあるとは、遷移先が自分自身でないこと、つまり遷移が起こることです。

問題設定テンプレ

  • 状態 \(X\) から遷移しうる \(X\) 以外の状態の集合を \(Y\) とする。
  • 一度の操作で \(Y\) への遷移が起こる確率を \(p\) とし、遷移が起こらない(\(X\) 自信に遷移する)確率を \(1-p\) とする。
  • \(Y\) は \(M\) 個の状態 \(S_1,S_2 , \dots , S_M\) からなる集合であるとする。
  • \(P(s)\) を、状態 \(X\) から状態 \(s\) に遷移する確率とします。
  • \(P(B|A)\) を、状態 \(X\) から状態 \(A\) に遷移したことが既に確定している時、その遷移先が状態 \(B\) である条件付き確率とします。

このような状況では、以下のテンプレで式を立てることができます。

・\(DP_X = \frac{1}{p} + \sum_{1 \leq i \leq M}(P(S_i|Y)\cdot DP_{S_i}) \)



これは、先ほど述べた式を使った結果です。
  • \(\frac{1}{p}\) = 確率 \(p\) の事象 (事象の集合\(Y\)) が起きるまで試行を行う回数の期待値
  • \(P(S_i|Y)\cdot DP_{S_i}\) = 遷移が起こった時、その遷移先が状態 \(S_i\) である場合の答えを \(DP_{S_i}\) を参照している

\(dp_{ (寿司が1の皿の枚数 , 2の枚数 , 3の枚数) }\) とすると、
\(dp_{(i,j,k)}\) の時、\(p = \frac{ i + j + k }{ N}\) である。
起こる事象としては、{ 1 の皿が選ばれる , 2 の皿が選ばれる , 3 の皿が選ばれる } のいずれかなので、
\(Y\) = {\( (i-1,j,k) , (i+1,j-1,k) , (i,j+1,k-1) \)} である。
これらを先ほどの式に当てはめれば良い 


int main(){
    ll n;
    cin >> n;
    double N = n;
    vector<vector<vector<double> > >
        dp(302,vector<vector<double> >
            (302,vector<double>
                (302,100000000)));
    vector<int> a(n);
    CIN(a);
    ll one = 0;
    ll two = 0;
    ll three = 0;
    REP(i,n){
        if(a[i] == 1)one++;
        if(a[i] == 2)two++;
        if(a[i] == 3)three++;
    }
    dp[0][0][0] = 0;
    REP(k,301){
        REP(j,301){
            REP(i,301){
                double S = (i+j+k);
                double p = S/N;
                if(p == 0)continue;
                dp[i][j][k] = 1/p;
                if(i!=0)dp[i][j][k] += dp[i-1][j][k]*(i/S);
                if(j!=0&&i<=300)dp[i][j][k] += dp[i+1][j-1][k]*(j/S);
                if(k!=0&&j<=300)dp[i][j][k] += dp[i][j+1][k-1]*(k/S);
            }
        }
    }
    
    cout.precision(15);
    cout << dp[one][two][three] << endl;
    return 0;
