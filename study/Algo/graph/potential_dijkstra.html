ポテンシャルダイクストラ

要編集記事です

ダイクストラ法は負辺があるグラフに適用することは出来ません。なぜなら、一度確定したと思った最短経路長が、負の辺を経由する別の経路によって更新されてしまう可能性があり、計算量が大きくなってしまうからです。

対して、ベルマンフォード法であれば単一始点最短経路問題を \(O(N×M)\) 時間で解くことができます。

さて、実は負辺があるようなグラフであっても、正しいポテンシャルを導入できればこの単一始点最短経路問題を \(O(NlogN)\) 時間で解くことができます。

ここで、頂点 \(s\) を始点とし、\(s\) から頂点 \(i\) までの最短経路長を \(d_i\) とします。

単一始点最短経路問題を解きたいグラフ G に頂点 u と 頂点 v を結ぶ辺が存在する時、その辺の重みを dist(u,v) とすると全ての辺 (u-v) について\(d_u + dist(u,v) \geq d_v\) が成り立ち、単一始点最短経路問題はこの制約の中で\( d_i \)を最大化するという線形計画問題として捉えられます。

最大化と書きましたが、\(d_i\) について、任意の u に対して \(d_u + dist(u,i) \geq d_i\) が成り立っているので、\(d_i\) を最大化するということは、ボトルネックとなる \(d_u + dist(u,i) \) (最小値) を選ぶということなので、最短経路問題との結びつきが見えました。

始点までの最短経路長は\( d_s = 0 \) とします

\(d_u + dist(u,i) \geq d_i\) の等号が成り立つとき、頂点 v までの最短経路に辺 u-v が含まれるということになります。

唐突ですが、ここで各頂点にポテンシャルというものを定めることにし、頂点 i の持つポテンシャルを\( p_i \)とします。

\(d_i = x_i + p_i\) と置くと、各辺に対して成り立つ不等式\(d_u + dist(u,v) \geq d_v\) は、\(x_u + p_u + dist(u,v) \geq x_v + p_v\)となります。これを少し式変形すると、\(x_u + dist(u,v) + p_u - p_v \geq x_v\) となるので、元のグラフ G の辺の重みを \(dist(u,v)\) から \(dist(u,v) + p_u - p_v \)に置き換えたグラフ H でダイクストラ法を行って単一始点最短経路問題を解くとで\( x_i \)を計算できます。ただし、dist(u,v) + p_i - p_j が負の値にならないように、適切に\(p_i\)の値を頂点 i に割り振る必要があります(後述)。

\(x_i\) がわかれば、\(d_i = x_i+p_i\) なので \(d_i\) を計算できるのですが、\(x_i\) も元のグラフ G と同じ辺構成(辺の重みのみ異なる)のグラフでの最短経路長なので、\(元のグラフ G での s までの最短経路長 = 0 = x_s\) のはずです。

しかし、\(d_i = x_i + p_i \) より、\(d_s = x_s + p_s = 0 + p_s = 0\) となるので \(p_i = 0 \)となってしまいます。

そこで、\(d_i\)を元のグラフ G の最短経路長として扱うのではなくて、\(x_i , p_i \)を計算した上で、\(d_i - p_s = x_i + p_i - p_s \)を元のグラフでの最短経路長として扱うと \(s までの最短経路長 = d_s - p_s = p_s - p_s = 0\)となるので、始点までの最短経路長が 0 であるという制約を満たします。

また、\(d_u + dist(u,v) \geq d_v\) の両辺から \(p_s\) を引くと、\(d_u - p_s + dist(u,v) \geq d_v - p_s\) となるので、\(d_i - p_s\) が単一始点最短系問題のもう一つの制約を満たしていることが分かります。

最後に、\(x_i\) を求めるために 元のグラフ G の辺の重みを \(dist(u,v)\) から \(dist(u,v) + p_u - p_v \)に置き換えたグラフ H での単一始点最短経路問題を解きたいので、これを O(NlogN) で解くために各辺について\(dist(u,v) + p_u - p_v \) が負にならないように、各頂点 i に適切なポテンシャル\( p_i \)を割り当てる必要があります。

ここで辺の重みが負にならないとは、グラフ H の任意の辺に対して、\(dist(u,v) + p_u - p_v \geq 0 \)が成り立つということなので、これを式変形すると\(p_u + dist(u,v) \geq p_v \)となります。

\( p_u + dist(u,v) \geq p_v \)とは、元のグラフ G での単一始点最短経路問題の制約なので、\(p_i\) は G の好きな地点を始点にする単一始点最短経路問題の回になります。dist(u,v) が負になることもあるので、\(p_i\) はベルマンフォード法によって O(N×M)で求めることができます。

これによって\( p_i \)が求まり、この\( p_i \)を使うことで負辺がある場合の単一始点最短経路問題を O(NlogN) で解ける様になりました。つまり、ポテンシャルダイクストラとは、O(N×M) かけて前計算しておくことで、負辺のある場合の単一始点最短経路問題を O(MlogN) で解ける様になるということです。
