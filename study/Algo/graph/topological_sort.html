トポロジカルソート


閉路のない有向グラフ \(G\) のトポロジカルソートとは、以下順序で頂点を並べた列を作ることです。

  • \(G\) の頂点 \(u , v\) について、\(u\) から \(v\) に到達可能ならば \(u\) は \(v\) より前の位置に、\(v\) から \(u\) に到達可能ならば \(u\) は \(v\) より後の位置に、どちらでもないなら \(u\) , \(v\) の順番は自由にする。



DAG 上で有向辺の 順/逆順 に DP したい場合などに役立ちます。

ただし、トポロジカルソートした数列で、u が v より前に現れても、必ずしもグラフ上で u から v に到達可能ではないです

実装も簡単で、頂点 \(x\) に入ってくる辺の本数を \(\mathrm{indegree}[x]\) とし、トポロジカル順で頂点を並べる数列を \(A\) とし、以下の手順を繰り返します。

  1. \(\mathrm{indegree}[x] = 0 \) となる頂点をすべて queue に入れる。
  2. queue が空になるまで、以下の 3,4 を繰り返す。
  3. queueから頂点を一つ取り出しそれを \(x\) として \(A\) の末尾に追加する。
  4. \(x\) から出る辺の先の頂点 \(n_1,n_2,n_3,\dots \) 全てに対して、\(\mathrm{indegree}[n_i]\) を \(1\) 減らして、\(0\) になったものを新たに queue に入れる。

最終的に得られる \(A\) がトポロジカルソートされた結果です。

このアルゴリズムは、\(G\) にサイクルがある場合にうまくいかないので、\(G\) が DAG かどうかの判定にも使うことができます。

\(G\) が DAG でない場合、トポロジカルソートされた配列 \(A\) のサイズと、頂点数が一致しないので、そこで判別します。

以下はコード例で、verifyはこちらです。 








/*トポソ*/
class TpSort{
    /*   
        Copyright ©️ (c) okoteiyu 2024. 
        Released under the MIT license(https://opensource.org/licenses/mit-license.php) 
    */
    protected:
    int m_V;//頂点数 (0 ~ m_V-1) 
    vector<vector<int>> m_G;

    // 頂点数は既に取得済みとする
    void init_member(){m_G = vector<vector<int>>(m_V);}
    public:
    TpSort(){}
    // 頂点数を指定
    TpSort(int V_ ):m_V(V_){init_member();}

    // 辺 (u->v) を追加
    void add_edge(int u , int v){m_G[u].push_back(v);}

    // トポロジカルソートを記録した配列を返す関数 
    // DAG でなければ、できたところまで返し、頂点数とサイズを比較して失敗かどうかを判定する
    template<typename Integer>
    vector<Integer> sorted(){
        vector<int> indegree(m_V,0);
        for(int i = 0 ; i < m_V ; i++)for(int j:m_G[i])indegree[j]++;
        vector<Integer> sorted_vertices;
        queue<Integer> q;
        for (int i = 0; i < m_V; i++) if (indegree[i] == 0)q.push(i);
        while (!q.empty()) {
            Integer u = q.front();
            q.pop();
            sorted_vertices.push_back(u);
            for(int v  : m_G[u]){
                indegree[v]--;
                if (indegree[v] == 0) q.push(v);
            } 
        }
        return sorted_vertices;
    }
    /*   
        Copyright ©️ (c) okoteiyu 2024. 
        Released under the MIT license(https://opensource.org/licenses/mit-license.php) 
    */
};
