添字を合わせるテクニック

プログラムを書いていて。配列やvectorの添字で頭がこんがらがってしまったことはありませんか？？ほくはあります。この記事では僕がいつも行っている添字の書き方を形式的にまとめてみようと思います。

  • step1 : とりあえずテキトーに添字を書く
  • step2 : 手元に簡単なケースを用意して、自分の書いたプログラム通りに処理する
  • step3 : 上で書いたテストケースの結果と、自分の期待する結果のズレの分だけ添字を修正する

例

数列 \(A\) があります。各 \( 1 \leq i \leq |A| \) について、A の \(i\) 番目から右に \(k\) 要素の和を出力してください。

まずはテキトーにコードを書きましょう。 

                    
vector<int> A;
                    
for(int i = 0 ; i <A.size() ; i++ ){
                    
    long long sum = 0;
                    
    for(int j = i ; j <= i+k ; j++ ){//赤字の部分、合ってるのかなぁ....
                    
        sum += A[j];
                    
    }
                    
    cout << sum << endl;
                    
}

そして、手元で長さ 5 ぐらいのテストケースを用意しましょう。今回は、\(A = 1,4,2,3,4\) , \(i = 0 , k = 3\) とします。

自分で書いたコード(上記のコード)通りに \(A\) を処理していき、for(int j = i ; j <= i+k ; j++ ) の部分に \(i\) , \(k\) を当てはめると \(j = 0 , 1 , 2 , 3 \)まで処理を繰り返すので、\(k\) 個の要素の和を求めたいはずが、 \(k+1\) 個の要素の和になっていておかしいと気付けます。

実際の結果と、期待する結果のズレが 1 だったので、\(j \leq i+k \) の部分を 1 ずらすと解決できそうですね。今回だと解答の修正部分は \( j \leq i + k - 1 \) になります。
