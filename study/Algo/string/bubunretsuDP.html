部分列DP

課題

以下のような問題を考えます。

文字列 \(S\) の部分列の種類数を \(\mathrm{mod}\space 998244353\) で答えなさい。

ただし \(S\) の部分列とは、\(S\) から \(0\) 個以上の文字を抜き出して、残った文字をその順に繋げた文字列のことです。

解法

以下の DP テーブルを構築します。

  • \(DP_i := \) 最初の文字が \(S\) の \(i\) (1-indexed) 文字目である部分列の種類数。
  • \(DP_0\) は空文字から始まる部分列、つまり、空文字列を含む全ての部分列の種類数を指します。

DP テーブルを埋める前に、以下の補助的なデータを用意します。

  • \( \mathrm{nex}_{{i,c}} :=\) \(S\) の \(i\) (1-indexed) 文字目よりも後で初めて文字 \(c\) が出てくる index
  • \( \mathrm{nex}_{{0,c}} :=\) \(S\) において、文字 \(c\) が初めて登場する index

また、\(nex_{i,c}\) の計算は、

  • \(S\) の \(i+1\) 文字目が \(c\) の場合 : \(\mathrm{nex}_{i,c} = i + 1\)
  • そうで無い場合: \(\mathrm{nex}_{i,c} = \mathrm{nex}_{i+1,c} \)
として、\(i\) が大きい順に計算できます。ただし、\(i\) などを 1-index で考えていることに注意してください。また、\(\mathrm{nex}\) の初期値には、解が存在しないことを表す \(\infty\) を採用すると直感的だと思います。

ここで、\(S_i = S_j\) である \( i , j \space(i \lt j)\) について、\(S_i\) から始めて生成可能な部分列 \(T_i\) は、\(S_j\) から始めて生成可能な部分列 \(T_j\) を含んでいます ( \(S_i\) から始めて、\(T_j\) の先頭より後の文字を \(T_j\) と同じように選ぶことで、\(T_j\) を構築できるから ) 。また定義より、\(DP_i\) の結果は \(DP_j\) もカウントしています。

さて、\(DP\) の遷移なのですが、\(DP_i\) は \(S_i\) のみの場合を含むので、\(DP_i = 1\) で初期化しておきます。

ここで、\(DP_i\) で数え上げる文字列 \(T_i\) は、\(T_i = S_i + T\prime \) という形の文字列で表せます。\(T\prime\) というのは、\(S_{i+1}\) 以降の文字を使った文字列です。

\(T\prime\) は \(S_{i+1}\) 以降の文字を使ってできる文字列なので、\(T\prime\) の数え上げは \(DP_j \space (i \lt j )\) を使って表せます。

\(S_i = S_j\) である \( i , j (i \lt j)\) について、\(DP_i\) の結果は \(DP_j\) を含んでいることと、先頭が異なる文字で始まる文字列同士は必ず異なるものであることから、\(T\prime\) としてカウントする文字列は、各文字 \(c\) ( 'a' ~ 'z' ) に対して、\(S_i\) の次に選ぶ文字が、\(i\) より後で初めて \(c\) が登場する箇所から始まる部分列に限定して良いです。

つまり、\(S_{i+1}\) 以降の文字から始まる文字列のうち、先頭の文字が異なるものを足し合わせることを考えます。

\(\mathrm{nex}\) を構築してあるので、このような部分列を以下の式でカウントできます。

  • \(DP_i = \sum DP[\mathrm{nex}_{i,c}]+1 \)

\(DP_i\) の計算には \(i\) より後ろの DP テーブルが必要なので、DP テーブルは \(i\) が大きい方から埋めていく必要があります。

最終的な答えは \(DP_0\) (空文字から始まる部分列の数え上げ) となります。 



int main(){
    string s;
    cin >> s;
    ll n = s.size();
    vector<vll> nex(n+2 , vll(26 , -1));
    for(int i = n - 1 ; i >= 0 ; i--){
        REP(c,26){
            if(s[i] == 'a' + c)nex[i][c] = i+1;
            else nex[i][c] = nex[i+1][c];
        }
    }
    vll dp(n+1,0);
    for(int i = n ; i>= 0 ; i--){
        dp[i] = 1;
        REP(c,26){
            if(nex[i][c] != -1)dp[i]+=dp[nex[i][c]];
        }
        dp[i]%=998244353;
    }
    cout << (dp[0]-1+998244353)%998244353 << endl;
