二分木

二分木は全ての頂点の子が高々 \(2\) である根付き木です。二分木の頂点同士の関係が以下の順序 (\(\leq\)) を満たすとき、この二分木を二分探索木と言います。

  • 「異なる頂点 \(a,b\) に対して \(a\leq b\)」\(\leftrightarrow\)「頂点 \(a\) が頂点 \(b\) より左の頂点である」
  • 任意の頂点 \(x\) について以下が成り立つ。
    • \(x\) の左の子の部分木から任意に頂点 \(y\) を選んだとき、\(x,y\) の順序は \(y\leq x\) である。
    • \(x\) の右の子の部分木から任意に頂点 \(y\) を選んだとき、\(x,y\) の順序は \(x\leq y\) である。
以上の順序に従って数列 \(a\) の要素を左から順に頂点に割り振ることで、二分探索木を数列と対応させることができます。なお、この記事では平衡二分木の使い道として、MultiSet Ver. と 数列 Ver. の \(2\) 種類を紹介します。


まずは二分木を数列と対応させた形で解説していきます。ここでは、平衡二分木の頂点は値を持つものとし、その値が数列の要素と対応します。

Tips

頂点に数列の要素を割り振ると説明しましたが、各頂点に index を明示的に持たせる必要はありません。というのも、後に説明する \(\mathrm{get}\) メソッドなどが説明するように、部分木サイズを参照することで要素アクセスや index の復元が可能だからです。

ただし、index を明示的に持たせる利点もあります。数列に要素が挿入/削除された場合も、index に対して区間加算を行うことで index の整合性を確保することができます。index の情報があるのであれば、以下の集約を持つことができます。

  • \(\mathrm{ProdSum} := \) 部分木以下の数列要素 \(S_i\) に対する \(\sum{i\times S_i}\)

木の平衡

二分探索木では、アクセスしたい頂点を根から探索します。このとき、木の深さが大きいと探索に時間がかかってしまいます。任意の葉頂点の深さの差が小さいとき、二分木が平衡であるといい、そのような二分木を平衡二分木と言います。



Splay

唐突ですが、「頂点にアクセスするたびに、その頂点が新たに木の根になるまで、木の順序を崩さないように回転する」操作を導入します。


意味がわからないと思いいますが、より具体的には、アクセスした頂点に対して以下の回転 (rotate) を行います。回転の方向は C が親 (P) から見て左右のどちらにあるかで異なります。また、C は P の親からの辺を引き継ぎます。


回転を繰り返すことで、アクセスした頂点はいずれ木の根になります。ここで更に以下の \(2\) 種類の回転を導入することで、なぜかだんだん木が平衡になります。


具体的には、自分 (アクセスした頂点) ,親,親の親 の位置関係で以下のように場合分けして回転を行います。

  • 親の親が存在しない場合、自分を \(1\) 回 rotate する。
  • (自分-親),(親-親の親) をつなぐ \(2\) 本の辺が同じ向きの場合、親を rotate してから自分自身を rotate する。
  • そうでない場合、自分を \(2\) 回 rotate する。
この操作を、アクセスした頂点が木の根になるまで繰り返します。これを splay操作 と言います。

集約と遅延評価

大体同じなので、より詳細な説明はセグ木の記事を参照してください。

集約

自身以下の部分木の要素の集約を保持することで、区間取得クエリに回答できます。要素アクセスのことを考えると、部分木に属する頂点の個数の集約は必須です。



遅延評価 ( lazy )

木を変形させて、ターゲットの区間をうまくカバーするように部分木を取り出すことで、部分木の要素を代表してその根に評価を遅延させることができます。そうすることで、区間更新や区間反転クエリに回答できます。以下の図は区間加算の遅延評価が存在する場合です。


評価 ( eval )

評価 (作用) が遅延しているノードを見たら、何よりもまず遅延評価を評価して解消します。評価した頂点から下のノードは未評価のままなので、評価後は子ノードに遅延評価を伝播させます。


集約の計算 ( update )

あるノードの集約は「左右のノードの集約 と 自分自身が持つ値 のマージ」です。ただし、左右のノードには遅延評価があるかもしれないので、先に左右のノードを評価 (eval) してから自身の集約を計算します。



rotate および 集約/遅延評価 の計算

rotate を行うと、辺の構成の変化によって頂点の 集約/遅延評価 が別のものにかかることなります。よって、rotate を行う前に少なくとも rotate による影響がある範囲内の遅延評価は解消し、また回転後は集約の値を再計算する必要があります。

以下の例は代表的な遅延評価の \(1\) 例である区間反転を持つ場合です。区間反転は、そのノード以降 (下側) のノードの順序を左右反転するという遅延評価です。


遅延評価は上から伝播するので、P \(\rightarrow\) C の順に評価(eval) を呼び出して遅延評価を解消します。


rotate の影響範囲内の遅延評価は解消されたので、C-P を回転し、その後集約を再計算します。

列操作 (split/merge)

split と merge をベースに列操作を行います。

まずは \(\mathrm{split}(x)\) を以下で定義します。

  • \(\mathrm{split}(x) := \) Splay Tree を、左から \(x\) 番目のノード以降と、それより前で \(2\) 分割する。
\(x\) 番目のノードを根としたとき、左の子につながる辺を cut します。このとき、分割された \(2\) つの Splay Tree の根をどこかに管理しておく必要があります。


また、split の逆の操作として \(\mathrm{merge}(r1,r2)\) を定義します。

  • \(\mathrm{merge}(r1,r2) := \) \(r1,r2\) を根とする Splay Tree をこの順に結合する。
\(r2\) の最左の頂点を splay し、その頂点の左の子として \(r1\) を link します。


split と merge は逆の操作です。split したものを merge して元に戻すとき、このことを利用して split は右から、merge は左から行うことで定数倍の改善が見込めます。

削除

splitで \(x\) 番目のノードと、その左右の木 \(L,R\) を切り離して、\(L,R\) のみを merge することで、木から \(x\) 番目のノードを削除できます。

要素の検索

もしもあるノードのポインタを持っているなら、そのノードが Splay Tree の左から何番目の要素か (0-index) を特定することができます。具体的には、そのポインタが指すノードを splay し、左の部分木のサイズを返すだけです。例えば、列型の Splay Tree には以下の機能を搭載できます。

  • \(\mathrm{find}(x) := \)「\(a_i = x\)」である \(i\)
    • 値が \(x\) であるノードへのポインタを、ハッシュテーブルなどで持っておく。
    • 区間更新と値の重複に対応していない。

上記の方法はシンプルですが、少し制約が強いのでもう少し工夫をして制約を緩和します。まず、要素の検索では区間更新がある場合の対応は厳しいです。しかし、値の重複に対しては以下のようにして対応できます。

  • \(\mathrm{find}(x,c) := \)「\(a_i = x\)」である \(i\) を \(c\) 種類選んで返す。ただし、選び方は保証しない。
    • 値が \(x\) であるノードへのポインタを管理する列を、以下のハッシュテーブルで持っておく。
      • \(\mathrm{ptr}[x] := \) 値に \(x\) を持つノードのポインタを管理する列。ただし、管理されているポインタの全てが有効とは限らない。
    • 要素の追加,要素の削除,値の変更 に対して、以下の方法で \(\mathrm{ptr}\) テーブルと連携させる
      • \(x\) をもつノードの追加 \(\rightarrow\) \(\mathrm{ptr}[x]\) にそのノードのポインタを追加する。
      • \(x\) をもつノードの削除 \(\rightarrow\) Splay Tree から削除後、そのノードの実体を破棄する代わりに削除を行った旨をそのノードのメンバに記録しておく。\(\mathrm{ptr}[x]\) への操作は行わない (ある種の遅延評価)。
      • あるノードが持つ値を \(x\) に変更する \(\rightarrow\) \(\mathrm{ptr}[x]\) にそのノードのポインタを追加する。元々そのノードへのポインタが管理されていた \(\mathrm{ptr}\) への操作は行わない (ある種の遅延評価)。
    • \(x\) を持つノード \(c\) 個の位置の計算を、以下の手続きで行う。
      • \(\mathrm{ptr}[x]\) の中身を順に見ていく。
      • チェックしたノードが以下のいずれかの場合は invalid なノードであるとして \(\mathrm{ptr}[x]\) から削除する。
        • ノードがすでに削除済みである。(ノードのメンバを確認して判断)
        • ノードの持つ値が \(x\) ではない。
        • すでに調べたものと同じノードである。(すでに見たノードのポインタをハッシュテーブルなどに記録)
      • \(c\) 種類列挙した時点で手続きを終了する。
    • あたりまえですが \(1\) つの Splay Tree クラスは、\(1\) つの連結成分を管理している前提が必要 (削除が記録されたノードは例外)。

invalid なノードを初めて見た時に削除することで、要素の削除,値の変更 の操作で \(\mathrm{ptr}\) に残された廃棄物をうまく弾くことができます。クエリが \(Q\) 回与えられ、うち \(\mathrm{find}\) で \(C\) 要素列挙する場合の計算量はクエリ全体で \(O((C+Q)\log{Q})\) 時間です。

また、\(\mathrm{pre}[x]\) を std::vector で実装すると格納したポインタの削除の計算量が良くありません。格納されている要素を前から見れば良いだけなので、代わりに連結リストで実装するなどが良いでしょう。


集約の例

ノードが持つ集約として、代表的なものとしては以下があります。

  • 自身以下の部分木のサイズ (\(\mathrm{SubTreeSize}\)) \(\rightarrow\) 左右の子の \(\mathrm{SubTreeSize}\) の和 \(+ 1\)(自身の分) で計算可能
  • 自身以下の部分木内のノードが持つ値の総和 (\(\mathrm{Sum}\)) \(\rightarrow\) 左右の子の \(\mathrm{Sum}\) の和 \(+ 1\)(自身の分) で計算可能
  • 自身以下の部分木内のノードが持つ値の最小値 (\(\mathrm{Min}\)) \(\rightarrow\) 左右の子の \(\mathrm{Min}\) と「自身が持つ値」の最小値から計算可能
  • 自身以下の部分木内のノードが持つ値の最大値 (\(\mathrm{Max}\)) \(\rightarrow\) 左右の子の \(\mathrm{Max}\) と「自身が持つ値」の最大値から計算可能

遅延評価の例

アフィン変換

\(\mathrm{affine}(x,a,b) := ax + b\) を、\(x\) に対するアフィン変換 \((a,b)\) と定義します。アフィン変換を遅延評価にすることで区間内の要素全てにアフィン変換を行うことができます。この記事で紹介した区間加算の遅延評価は、アフィン変換の特殊な場合です (\(b\) 加算は、\((1,b)\) と同じ)。また、区間内の値を全て \(x\) にする区間代入は、\((0,x)\) の区間アフィン変換と同じです。

\(x\) に対するアフィン変換 \(\mathrm{affine}(x,a,b)\) は、\(2\) つのパラメータ \((a,b)\) で決定されるので、この \((a,b)\) を遅延評価として保持します。既に \((a_1,b_1)\) の遅延評価を持つノードに新たに遅延評価 \((a_2,b_2)\) が加わるとき、二つの遅延評価をマージした \(({a_1}{a_2} , {a_2}{b_1} + b_2)\) を新たな遅延評価として保持させることができます。

集約への影響

遅延したアフィン変換 \(a,b\) の評価は、ノードが持つ値に対してアフィン変換を適用するだけでなく、ノードが持つ集約値にも適用する必要があります。先述の集約の例について、どのように評価すべきかを説明します。

  • \(\mathrm{SubTreeSize} \rightarrow \) アフィン変換の影響はない
  • \(\mathrm{Sum}\) に \(a\times\mathrm{Sum} + b\times\mathrm{SubTreeSuze}\) を代入する (部分木全てに \(b\) 加算するので \(\mathrm{SubTreeSize}\) をかける)
  • \(\mathrm{Min} , \mathrm{Max}\) について
    1. \(\mathrm{Min},\mathrm{Max}\) にそれぞれ \(a\times\mathrm{Min} + b\) と \(a\times\mathrm{Max} + b\) を代入する。
    2. アフィン変換後、\(\mathrm{Max} \leq \mathrm{Min}\) なら、\(\mathrm{Min}\) と \(\mathrm{Max}\) を入れ替える ( \(a\) の符号の影響 )。
    \(\mathrm{Min} , \mathrm{Max}\) のどちらか一方しか集約しない場合はアフィン変換を行えないことに注意してください。

区間反転

rotate で例示したように、遅延評価によって区間内の要素の順番の左右反転を行うことができます。あるノードの反転の遅延評価の評価時に左右の子の参照を入れ替え、子に反転の遅延評価を伝播させることで、最終的に葉まで反転の評価を到達させます。また、既に反転の遅延評価をもつノードに新たに反転の遅延評価を加えると、それらは打ち消しあって反転の評価が解消されます。


MultiSet にする

これまで説明してきた平衡二分木は、頂点を数列の要素に対応させたものでした。ここで、管理する数列 \(a\) に以下の制約がついているとき、平衡二分木を順序付き多重集合として使用することができます。

  • \(i\leq j \rightarrow a_i \leq a_j\)
MultiSet として扱う平衡二分木も、当然先で紹介した操作を行うことができます。


MultiSet 型の平衡二分木は特殊なケースの数列 (ソートされた数列) に対応しているので、先に挙げた操作に加えてさらにいくつかの操作を行うことができます。

領域の分割

  • \(\mathrm{lower\_bound}\prime(x) := \) \(x\) 以上の要素のうち、最も左のものを検索する。
  • \(\mathrm{upper\_bound}\prime(x) := \) \(x\) より大きいの要素のうち、最も左のものを検索する。
要素アクセスと同様に、根から降りながら探索していきます。今いる頂点が持つ値と \(x\) の大小を比較して降りるべき方向を決めます。

要素の位置に対して、上記の \(x\) に関する条件に単調性があることに注目すると、 \(\mathrm{lower\_bound}\prime(x)\) や \(\mathrm{upper\_bound}\prime(x)\) で求めるものはいずれも、\(x\) に関する条件で領域を二分したとき、その境界の直後であることがわかります。

要素の発見,削除

\(\mathrm{lower\_bound}(x)\) から、以下の機能を追加できます。

  • \(\mathrm{find}(x) := \) \(x\) が小さい方から何番目の要素かを返す。存在しなければ \(-1\) を返すなど。
  • \(\mathrm{erase}(x) := \) \(x\) を多重集合から \(1\) つ削除する。
これだけで C++ の std::set の完全上位互換と言えますし、集約などを活用することでさらに機能を追加できます。例えば頂点にペア \((k,v)\) を持たせて、\(k\) の範囲が \([l,r)\) 以内の要素の \(v\) の総和や Min/Max を取ったりもできます (要素の順序は \((k,v)\) の辞書順)。

            |- 半開区間 [l,r) の要素を reverse することができる (区間反転)
                |- _use_find_ が true の場合の計算量解析不明 , 現状、落ちるケースもある
            |- 区間 [l,r) を c 回 circular_shift できる (左右)
                |- スロットのリールのように c 回回転させる
                |- c が負の場合は左回転になる。いずれの場合も、一周以上回る場合はエラー (外で処理して)。
                |- _use_find_ が true の場合の計算量解析不明 , 現状、落ちるケースもある
            |- 区間内の要素に一律に更新をかけることができる
                |- _use_find_ が true なのに更新をかけると RE になる。
                |- 区間内の要素に対して、代入/アフィン変換/chmax など
    |
    - 高速化について 
        |- 遅い時はいらない集約や遅延評価を消す。
        |- 遅ければデストラクタも消そう
    |
    - ライブラリの設計 : 
        |- 内部処理は内部処理で完結するように (内部処理は木の構造を触るため繊細)。
        |- 抽象化はせずに、モノイド積や遅延評価作用を例示することで、書き換えを容易にする。  
            |- Sum , Min , Max , RangeUpdate , RangeAffine , RangeReverse で特殊化している
            |- 不必要なモノイド積や作用の記述を消すと速くなる  
        |- 特殊化されたモノイド積や作用は以下に書き込まれているので、書き換えるときは主に以下を書き換える
            |- Node のメンバ (lazy_affine や set_LazyAffine)
            |- Node の update ,eval の処理
        |- get(i) や getRange(l,r) で取得するノードは、要素取得の用途での使用のみを想定しているので、
            / 隣接するノードへのアクセスを封印したコピーを返すようにしてある。
*/
template<class T , bool _use_find_ = false >
class MyBinaryVector{
    /*

        Released under the MIT license(https://opensource.org/licenses/mit-license.php) 
    */


    private:
    struct SplayNode{
        friend class MyBinaryVector;

        public:

        T Value;// ノードが持つ値
        T Sum;  // 部分木要素の和
        T Max;  // 部分木要素の Max
        T Min;  // 部分木要素の Min
        
        int SubTreeSize = 1; // 部分木のサイズ

        private:

        SplayNode *parent = nullptr; //親ノード
        SplayNode *left = nullptr;   //左の子ノード
        SplayNode *right = nullptr;  //右の子ノード

        inline static const pair<T,T> eq_affine = {T(1),T(0)}; // アフィン変換の単位元
        pair<T,T> lazy_affine = eq_affine;// 遅延中のアフィン変換のパラメータ

        /*==================================================
            メモリアライメントに配慮して、bool は最後に宣言する
        ===================================================*/

        // 反転が遅延しているかどうか
        bool lazy_reverse = false;
        // ノードのコピーかどうか ( splay tree の get などに詳細 )
        bool temp_instance = false;
        // このノードがどの木にも属さなくなったか(削除されたか)。
        bool isThisDeleted = false;
        
        SplayNode(){}
        SplayNode(T val) : Value(val){
            update();
        }

        // デフォルトのシャローコピーを採用
        SplayNode(const SplayNode&) = default;
        SplayNode(SplayNode&&) = default;
        SplayNode& operator=(const SplayNode&) = default;
        SplayNode& operator=(SplayNode&&) = default;

        // データ読み込み用の、隣接頂点へのアクセスを禁止したコピーを返す。
        SplayNode make_temp_instance(){
            assert(temp_instance == false);
            SplayNode res = *this;
            res.left = nullptr;
            res.right = nullptr;
            res.parent = nullptr;
            res.temp_instance = true;
            return res;
        }
        

        // 反転を遅延させる
        inline void set_lazyReverse() noexcept {
            this->lazy_reverse = !(this->lazy_reverse);
        }

        // アフィン変換を遅延させる
        inline void set_lazyAffine(T a, T b) noexcept {
            this->lazy_affine = {a*this->lazy_affine.first , a*this->lazy_affine.second + b};
        }
        

        // this - this->parent を回転させる (親は必ず存在するものとし、位置関係も誤りがないものとする)
        void rotate() noexcept {
            // GrandParentの子を張り替え
            if(this->parent->parent != nullptr) [[likely]] {
                if(this->parent == this->parent->parent->left)this->parent->parent->left = this;
                else this->parent->parent->right = this; 
            }
            
            this->parent->eval();
            this->eval();
            
            //元の親に対する位置で場合分け
            if(this->parent->left == this){
                this->parent->left = this->right;
                if(this->right != nullptr)this->right->parent = this->parent;
                this->right = this->parent;
                this->parent = this->right->parent;
                this->right->parent = this;
                this->right->update();
            }else{
                this->parent->right = this->left;
                if(this->left != nullptr)this->left->parent = this->parent;
                this->left = this->parent;
                this->parent = this->left->parent;
                this->left->parent = this;
                this->left->update();
            }

            this->update();
            return;
        }
        /*
            自分から見た親の位置
            0 -> 親が存在しない
            1 -> 親の左の場合
            2 -> 親の右の場合
        */
        int state() noexcept {
            if(this->parent == nullptr) [[unlikely]] return 0;
            this->parent->eval();
            if(this->parent->left == this)return 1;
            return 2;
        }

        // あるNodeを回転を駆使し一番上まで持っていく
        void splay() noexcept {
            while(this->parent != nullptr){
                if(this->parent->state() == 0){
                    this->rotate();
                    break;
                }
                // 親の state をすでに計算しているので、親の親 → 親 → 自分に reverse の評価を降ろせている
                if( this->parent->state() == this->state() )this->parent->rotate();
                else this->rotate();
                this->rotate();
            }
            this->update();// while が呼ばれなかった場合のため
            return;
        }

        // Nodeの持っている情報をupdateする
        void update() noexcept {
            assert(temp_instance == false);

            this->eval();
            this->Sum = this->Value;
            this->Max = this->Value;
            this->Min = this->Value;
            this->SubTreeSize = 1;
            if(this->left != nullptr) [[likely]] {
                this->left->eval();
                this->SubTreeSize += this->left->SubTreeSize;
                if(this->left->Min < this->Min)this->Min = this->left->Min;
                if(this->left->Max > this->Max)this->Max = this->left->Max;
                this->Sum += this->left->Sum;
            }
            if(this->right != nullptr) [[likely]] {
                this->right->eval();
                this->SubTreeSize += this->right->SubTreeSize;
                if(this->right->Min < this->Min)this->Min = this->right->Min;
                if(this->right->Max > this->Max)this->Max = this->right->Max;
                this->Sum += this->right->Sum;
            } 
            return;
        }

        /*
            この頂点にアクセスされるたびに、各flagがtrueなら遅延させていた処理を実行し、左右の子にflagを伝播する
                - reverse : 自分の左右の子を入れ替えて、左右の子に遅延伝播(最終的に区間全体が反転する)
                - affine : 自分自身以下の部分木のValueをアフィン変換する (A 掛けて B 足す) 
                - Min,Max,Sum なども、遅延している値から計算する
        */
        void eval() noexcept {
            assert(temp_instance == false);
            
            if(this->lazy_reverse){
                swap(this->left , this->right);
                //下に伝播
                if(this->left != nullptr)this->left->set_lazyReverse();
                if(this->right != nullptr)this->right->set_lazyReverse();
                this->lazy_reverse = false;
            }

            // アフィン変換
            if(this->lazy_affine != eq_affine){
                this->Value = this->lazy_affine.first * this->Value + this->lazy_affine.second;
                this->Min = this->lazy_affine.first * this->Min + this->lazy_affine.second;
                this->Max = this->lazy_affine.first * this->Max + this->lazy_affine.second;
                this->Sum = this->lazy_affine.first * this->Sum + this->SubTreeSize*this->lazy_affine.second;
                if(this->Max < this->Min)swap(this->Max,this->Min);
                
                if(this->left != nullptr)this->left->set_lazyAffine(this->lazy_affine.first,this->lazy_affine.second);
                if(this->right != nullptr)this->right->set_lazyAffine(this->lazy_affine.first,this->lazy_affine.second);
                // 遅延評価を解消
                this->lazy_affine = eq_affine;
            }
        }
    };



    protected:
    /*=================================
                メンバ変数たち
    =================================*/

    // 常に m_Root が根であるようにする (merge した後は m_Root に代入するなど)
    SplayNode *m_Root = nullptr;
    
    /*
        - m_pointers[x] := 値が x であるようなSplayNodeのポインタ達。
            |- insert , update_val が呼ばれた時に変更する。
            |- pop などで削除があって m_pointers から即座に削除はせず、find() で中身を見るついでに削除する。
            |- m_pointers[x] に同じノード(ポインタ)が複数存在する場合や、x 以外の値を持つノードのポインタがある場合がある。
        - find(x) で m_pointers[x] の中身を見るついでに、複数存在するものや x 以外のものを削除する。
    */
    map<T , list<SplayNode*> > m_pointers; 


    private:


    // rootを根とする部分木において、求めたいindexまで降りていく(左側の部分木サイズを参照する)
    static SplayNode *get_sub(int index, SplayNode* root){
        if(root == nullptr)return root;
        assert(root->parent == nullptr);

        auto now = root;
        while(true){
            now->eval();// ノードを見る前にeval
            int left_size = bool(now->left == nullptr) ? 0 : now->left->SubTreeSize;
            if(index == left_size)break;
            if(index < left_size){
                now = now->left;
            }else{
                now = now->right;
                index -= left_size+1;
            }
        }
        now->splay();
        return now;
    }
 

    /*==============================================================================
        merge/split ベースの実装を行う。
        |- merge(lef,rig) : rig の左の子に lef をつける
        |- split(i,rt) : i 番目のノードの左の辺を無くし、(lef,rig) のペアを返す
        merge と split が反対の操作であることを利用して計算量を削減する
    ==============================================================================*/

    
    // leftroot を根とする SplayTree と、rightroot を根とする SplayTree をこの順に結合する。
    static SplayNode *merge(SplayNode *leftRoot, SplayNode *rightRoot){
        if(leftRoot == nullptr)return rightRoot;// 右のみ
        if(rightRoot == nullptr)return leftRoot;// 左のみ
        assert(leftRoot->parent == nullptr && rightRoot->parent == nullptr);
        // 怖いのでここで update かけときます。大したコストではないです 
        leftRoot->update();
        rightRoot->update();

        rightRoot = get_sub(0,rightRoot);
        rightRoot->left = leftRoot;
        leftRoot->parent = rightRoot;
        rightRoot->update();
        return rightRoot;
    }
    
    // root が根の SplayTree を、左から i-1 番目と i 番目の間で分割する。
    std::pair<SplayNode*,SplayNode*> split(int i, SplayNode*root){
        if(root == nullptr)return std::make_pair(nullptr , nullptr);
        assert(root->parent == nullptr);
        assert(0 <= i && i <= root->SubTreeSize);
        if(i == 0)return std::make_pair(nullptr , root);
        if(i == root->SubTreeSize)return std::make_pair(root, nullptr);
        root = get_sub(i , root);
        auto leftRoot = root->left;
        auto rightRoot = root;
        if(rightRoot != nullptr)rightRoot->left = nullptr;
        if(leftRoot != nullptr)leftRoot->parent = nullptr;
        leftRoot->update();
        rightRoot->update();
        return std::make_pair(leftRoot,rightRoot);
    }


    public:

    MyBinaryVector() : m_Root(nullptr) , m_pointers(){}

    // std::vector スタイルの初期化
    MyBinaryVector(const vector<T>& array_) : MyBinaryVector() {
        if(array_.size() == 0)return;
        for(T x : array_)this->push_back(x);
    }

    MyBinaryVector(int size_ , T init_value) : MyBinaryVector(vector<T>(size_, init_value)){}


    ~MyBinaryVector(){
        // メモリ解放。MyBinaryVector のような要素検索がある場合は、多重解放にならないようにしてね
        while(m_Root != nullptr)this->pop(0);
    }

    MyBinaryVector(const MyBinaryVector<T,_use_find_>& param_tree) : MyBinaryVector() {
        // subr 以降の部分木をコピー
        function<SplayNode*(const SplayNode*)>subr_copy = [&](const SplayNode* subr){
            if(subr == nullptr)return static_cast<SplayNode*>(nullptr);
            SplayNode* res = new SplayNode(*subr);
            res->left = subr_copy(res->left);
            res->right = subr_copy(res->right);
            if(res->left != nullptr)res->left->parent = res;
            if(res->right != nullptr)res->right->parent = res;
            return res;
        };
        m_Root = subr_copy(param_tree.m_Root);

        // m_pointer に入れていく
        if(_use_find_ && m_Root != nullptr){
            stack<SplayNode*> st;
            st.push(m_Root);
            while(!st.empty()){
                auto now = st.top();
                st.pop();
                now->eval();
                m_pointers[now->Value].push_back(now);
                if(now->left != nullptr)st.push(now->left);
                if(now->right != nullptr)st.push(now->right);
            }
        }
    }
    
    // ムーブ
    MyBinaryVector(MyBinaryVector<T,_use_find_>&& param_tree) noexcept : MyBinaryVector() {
        swap(this->m_Root, param_tree.m_Root);
        this->m_pointers.swap(param_tree.m_pointers);
    }

    // コピー&スワップ イディオム
    MyBinaryVector& operator=( MyBinaryVector<T,_use_find_> param_tree) noexcept {
        swap(this->m_Root, param_tree.m_Root);
        this->m_pointers.swap(param_tree.m_pointers);
        // param_tree に渡したリソースは param_tree のデストラクタで破棄される
        return *this;
    }

    // 要素数
    int size() noexcept {
        if(m_Root == nullptr)return 0;
        return m_Root->SubTreeSize;
    }


    // 左から i 番目のNodeのコピーを取得 (0-index)
    // ただし、値の取得が目的なので right などの隣接頂点へのアクセスを封印する
    SplayNode get(int i){
        assert(0 <= i && i < size());
        m_Root = get_sub(i,m_Root);
        return m_Root->make_temp_instance();
    }


    // [l,r) に対応する部分木の根のコピーを返す。ただし、隣接ノードへのアクセスを封じる。
    // 使用用途はモノイド積の取得に限定 (コピーなので、オリジナルの現在の値とは関係なく [l,r) のデータが格納されている)
    SplayNode GetRange(int l , int r){
        assert(0 <= l && l < r && r <= size());
        auto tmp = split(r,m_Root);
        auto rightRoot = tmp.second;
        tmp = split(l,tmp.first);// 部分木を取り出す。
        auto res = tmp.second->make_temp_instance();
        m_Root = merge(merge(tmp.first,tmp.second),rightRoot);
        return res;
    }

    // index 番目に x を挿入
    void insert(int index , T x){
        assert(0 <= index && index <= size());
        SplayNode* NODE = new SplayNode(x);
        NODE->update();

        if(m_Root == nullptr)m_Root = NODE;
        else{
            auto Trees = split(index,m_Root);
            m_Root = merge(merge(Trees.first,NODE),Trees.second);
        }
        // 位置検索用ポインタリストに追加
        if(_use_find_)m_pointers[NODE->Value].push_back(NODE);
    }
    
    // index番目のNodeを消去
    void pop(int index){
        assert(0 <= index && index < size());
        auto center = get_sub(index,m_Root);
        auto leftRoot = center->left;
        auto rightRoot = center->right;
        if(leftRoot != nullptr)leftRoot->parent = nullptr;
        if(rightRoot != nullptr)rightRoot->parent = nullptr;
        center->left = nullptr;
        center->right = nullptr;
        center->update();
        center->isThisDeleted = true;
        if(_use_find_ == false)delete center;

        m_Root = merge(leftRoot,rightRoot);
    }

    // x 番目のvalueをyにする
    void update_val(int x , T y){
        assert(0 <= x && x < size());
        m_Root = get_sub(x,m_Root);
        m_Root->Value = y;
        m_Root->update();// ノードを編集したら update!!
        if(_use_find_)m_pointers[y].push_back(m_Root);
    }

    // [l,r)を反転
    void reverse(int l , int r){
        assert(0 <= l && l < r && r <= size());
        if(r-l <= 1)return;

        auto tmp = split(r,m_Root);
        auto rightRoot = tmp.second;
        tmp = split(l,tmp.first);
        tmp.second->set_lazyReverse();
        tmp.second->update();
        m_Root = merge(merge(tmp.first, tmp.second),rightRoot);
        return;
    }

    // 木全体の[l,r)に区間アフィン変換 (区間内の要素全てに、A 掛けて B 足す)
    void RangeAffine(int l , int r , T A , T B){
        assert(!_use_find_);
        assert(0 <= l && l < r && r <= size());

        auto tmp = split(r,m_Root);
        auto rightRoot = tmp.second;
        tmp = split(l,tmp.first);
        tmp.second->set_lazyAffine(A,B);
        tmp.second->update();
        m_Root = merge(merge(tmp.first, tmp.second),rightRoot);
        return;
    }

    // 木全体の[l,r)に区間加算(区間内の要素全てに+x) ((1,x) のアフィン変換と同値)
    void RangeAdd(int l , int r , T x){
        assert(!_use_find_);
        assert(0 <= l && l < r && r <= size());
        RangeAffine(l,r,T(1),x);
        return;
    }
    // 木全体の[l,r)に区間更新(区間内の要素全てをxにする)
    void RangeUpdate(int l , int r , T x){
        assert(!_use_find_);
        assert(0 <= l && l < r && r <= size());
        RangeAffine(l,r,T(0),x);
        return;
    }

    // 半開区間 [l,r) の要素を、右に c 回 circular shift する。
    // つまり、スロットのリールのように c 回右に回転させる
    // c が負の場合は左回転になる。いずれの場合も、一周以上回る場合はエラー (外で処理して)。
    void circular_shift(int l , int r , long long c){
        assert(0 <= l && l < r && r <= size());
        if(abs(c) == r - l)return;
        assert(c < abs(r-l));
        if(c == 0)return;
        if(c < 0)c = r-l+c;

        // 以降は 0 < c < (r-l) である
        auto tmp = split(r,m_Root);
        auto rightRoot = tmp.second;
        tmp = split(l,tmp.first);// 部分木を取り出す。
        auto leftRoot = tmp.first;
        auto center = tmp.second;
        tmp = split(r-l-c , center);
        
        m_Root = merge(merge(merge(leftRoot,tmp.second),tmp.first),rightRoot);
        return;
    }


    // 数列内の x の位置を vector<int> に入れて返す。
    // count で指定した個数以下の個数(選び方は未定義)を返す
    vector<int> find(T x , int count){
        assert(_use_find_);
        if(size() == 0)return vector<int>(0);
        if(m_pointers[x].size() == 0)return vector<int>(0);
        vector<int> res;
        unordered_map<SplayNode*,bool> checked;

        // m_pointers[x] を一つずつ見ていく
        // erase があるので、後ろから見ていく
        auto it = m_pointers[x].end();
        while(it != m_pointers[x].begin()){
            if(res.size()>=count)break;
            auto Nd = *(--it);// end を見ないように、先にデクリメント
            assert(Nd != nullptr);
            
            // erase(it) は it を削除し、次の要素のイテレータを返す
            if(checked[Nd])it = m_pointers[x].erase(it);// 一度見たものは問答無用で弾く
            else if(Nd->isThisDeleted)it = m_pointers[x].erase(it); //削除済みを弾く
            else if(Nd->Value != x)it = m_pointers[x].erase(it); //値の変更ずみを弾く
            else{ 
                // この時点で、 Nd は実在してかつ初めて見るノードであることが確定している
                Nd->splay();
                m_Root = Nd;
                int index = 0;
                if(Nd->left != nullptr)index+=Nd->left->SubTreeSize; //同じ NODE へのポインタが複数存在する場合はpop
                res.push_back(index);//見つけた場合、答えに追加して、checkedでメモ    
            }
            // 既に見たかを記録。nullptr も含めて記録して OK
            checked[Nd] = true;
        }
        return res;
    }

        std::cerr<< std::endl;
    }

/*

    - ノードが type_key 型のKey と type_value 型のValueを持つ Splay Tree で、(Key , Value) が +++ 辞書順 +++ に並んでいる
        |- C++ の set とは異なり、Key の重複を許す。(Value も当然重複 OK)
    - get(i) や [i] で i 番目のノードのコピーを取得 (隣接頂点へのアクセス (ポインタ) が封印されたものを返す) (0-index)
    - GetRange(l,r) は半開区間 [l,r) をカバーする部分木の根のコピーを返す。get 同様に、隣接頂点のポインタは封印されている。
        |- GetRange(l,r).Sum_val のようにして [l,r) の持つ要素のモノイド積を取得する
    - (Key,Value) に関して以下の操作が可能
        |- insert_pair(k,v) で (k,v) を持つノードを追加し、erase_pair(k,v) で (k,v) を持つノードを(存在すれば)削除する。
        |- upper_bound_pair(k,v) : (Key,Value) が辞書順で (k,v) 以下の要素数を返す
        |- lower_bound_pair(k,v) : (Key,Value) が辞書順で (k,v) 未満の要素数を返す
        |- count_pair(k,v) : (Key,Value) が (k,v) である要素の個数
        |- find_pair(k,v) : (Key,Value) が (k,v) である要素の index を返す。存在しなければ -1 を返す。(0-index)
    - ノードの Key だけに注目して、通常の set のように振る舞わせることもできる。  
        |- insert(k) : Key = k である要素を追加する。ただし、Value を指定しないので Value は未定義とする。
        |- erase(k) : Key = k である要素を一つ削除する。ただし、Value について特に指定しないことに注意。
        |- upper_bound(k) : Key が k 以下の要素数を返す
        |- lower_bound(k) : Key が k 未満の要素数を返す
        |- count(k) : Key が k である要素の個数
        |- find(k) : Key が k である要素の index を返す。存在しなければ -1 を返す。(0-index)
    - ペアを持つノードとKeyしか持たないノードが混在するといけないので、insert(k) について以下の仕様とする。
        |- insert() を呼び出した時点で、Value が未定義の要素が存在することになるので
            // upper_bound_pair , find_pair などの
            // Value に関する関数を呼び出すとランタイムエラーになるようにしました。
            // m_paired というメンバで管理してます
        |- Key だけに関係する関数は変わらず使用できる。
    - SplayNode の < 演算をを変更することで要素の並び順を変更できる。
    - Splay Tree ライブラリの仕様&設計 : 
        |- 関数に渡す SplayNode や、関数から返される SplayNode は全て update (& eval) されている必要がある
        |- 平衡二分木は、アクセスするだけでも構造を変えるので、const 修飾できない。
        |- get(i) や getRange(l,r) で取得するノードは、要素取得の用途での使用のみを想定しているので、
            / 隣接するノードへのアクセスを封印したコピーを返すようにしてある。
    - 使う際の注意点
        |- 遅延評価を載せる場合は色々注意
            |- 変更後の値が sorted になるような制限をつけないとダメ
*/
template<class type_key , class type_value>
class MyMultiSet{
    /*   
        Copyright © (c) okoteiyu 2024. 
        Released under the MIT license(https://opensource.org/licenses/mit-license.php) 
    */

    private:
    struct SplayNode{
        friend class MyMultiSet;
        
        public:
        
        type_key Key;       // 値 (この値をもとに昇順に並んでいる)
        type_key Sum_key;   // 部分木のノードの Key の値の和

        type_value Value;   // Key とは別に頂点に持たせる値
        type_value Sum_val; // 部分木のノードの val の値の和
        type_value Max_val; // 部分木のノードの val の値の Max
        type_value Min_val; // 部分木のノードの val の値の Min
        
        int SubTreeSize = 1;// 部分木サイズ

        private:
        
        SplayNode *parent = nullptr;  // 親ノード
        SplayNode *left = nullptr;    // 左の子ノード
        SplayNode *right = nullptr;   // 右の子ノード
            
        // ノードのコピーかどうか ( splay tree の get などに詳細 )
        bool temp_instance = false;

        
        SplayNode(){}
        SplayNode(type_key key_ , type_value val_) : Key(key_), Value(val_){
            update();
        }
        
        // デフォルトのシャローコピーを採用
        SplayNode(const SplayNode&) = default;
        SplayNode(SplayNode&&) = default;
        SplayNode& operator=(const SplayNode&) = default;
        SplayNode& operator=(SplayNode&&) = default;


        // データ読み込み用の、隣接頂点へのアクセスを禁止したコピーを返す。
        SplayNode make_temp_instance(){
            assert(temp_instance == false);
            SplayNode res = *this;
            res.left = nullptr;
            res.right = nullptr;
            res.parent = nullptr;
            res.temp_instance = true;
            return res;
        }
        
        
        // this - this->parent を回転させる (親は必ず存在するものとし、位置関係も誤りがないものとする)
        void rotate() noexcept {
            //GrandParentの子を張り替え
            if(this->parent->parent != nullptr) [[likely]] {
                if(this->parent == this->parent->parent->left)this->parent->parent->left = this;
                else this->parent->parent->right = this; 
            }

            this->parent->eval();
            this->eval();
            
            //元の親に対する位置で場合分け
            if(this->parent->left == this){
                this->parent->left = this->right;
                if(this->right != nullptr)this->right->parent = this->parent;
                this->right = this->parent;
                this->parent = this->right->parent;
                this->right->parent = this;
                this->right->update();
            }else{
                this->parent->right = this->left;
                if(this->left != nullptr)this->left->parent = this->parent;
                this->left = this->parent;
                this->parent = this->left->parent;
                this->left->parent = this;
                this->left->update();
            }

            this->update();
            return;
        }

        /*
            自分から見た親の位置
            0 -> 親頂点がない
            1 -> 親の左の場合
            2 -> 親の右の場合
        */
        int state() noexcept {
            if(this->parent == nullptr) [[unlikely]] return 0;
            this->parent->eval();
            if(this->parent->left == this)return 1;
            return 2;
        }

        // あるNodeを回転を駆使し一番上まで持っていく
        void splay() noexcept {
            while(this->parent != nullptr){
                if(this->parent->state() == 0){
                    this->rotate();
                    break;
                }
                if( this->parent->state() == this->state() )this->parent->rotate();
                else this->rotate();
                this->rotate();
            }
            this->update();// while が呼ばれなかった場合のため
            return;
        }

 
        // サイズなどの情報をupdateする
        void update() noexcept {
            assert(temp_instance == false);
            
            this->eval();
            this->SubTreeSize = 1;
            this->Sum_key = this->Key;
            
            this->Sum_val = this->Value;
            this->Max_val = this->Value;
            this->Min_val = this->Value;
            
            if(this->left != nullptr) [[likely]] {
                this->left->eval();
                this->SubTreeSize += this->left->SubTreeSize;
                if(this->left->Min_val < this->Min_val)this->Min_val = this->left->Min_val;
                if(this->left->Max_val > this->Max_val)this->Max_val = this->left->Max_val;
                this->Sum_key += this->left->Sum_key;
                this->Sum_val += this->left->Sum_val;
            }
 
            if(this->right != nullptr) [[likely]] {
                this->right->eval();
                this->SubTreeSize += this->right->SubTreeSize;
                if(this->right->Min_val < this->Min_val)this->Min_val = this->right->Min_val;
                if(this->right->Max_val > this->Max_val)this->Max_val = this->right->Max_val;
                this->Sum_key += this->right->Sum_key;
                this->Sum_val += this->right->Sum_val;
            }
 
            return;
        }

        // 遅延評価を評価する (基本は不要なので不要なので空にしておく)
        void eval() noexcept {
            assert(temp_instance == false);
        }

        // ノードの値によって順序を定義。ただし、共に遅延評価がなく、正しい情報を持っている必要がある。
        // {Key,Value} の辞書順
        friend bool operator<(const SplayNode& a , const SplayNode& b) noexcept {
            if(a.Key != b.Key)return bool(a.Key < b.Key);
            return bool(a.Value < b.Value);
        }

        
    };

    
    protected:
    /*=================================
                メンバ変数たち
    =================================*/
                
    
    // 管理する木の根ノード
    SplayNode *m_Root = nullptr;
    // upper_bound などで比較を行うためのノードオブジェクト。SplayTree に挿入はしない
    SplayNode* const m_bluff_object = new SplayNode();
    // (Key,Value) の両方が有効な値かどうか。insert(k) を呼び出すと未定義の Value が含まれることになるので false になる。
    bool m_paired = true;
    // Value を指定しない場合のデフォルト値
    inline static const type_value m_default_val = type_value(0);
    
    private:
    
    // rootを根とする部分木において、求めたいindexまで降りていく(左側の部分木サイズを参照する)
    static SplayNode *get_sub(int index, SplayNode* root){
        if(root == nullptr)return root;
        assert(root->parent == nullptr);

        auto now = root;
        while(true){
            now->eval();// ノードを見る前にeval
            int left_size = bool(now->left == nullptr) ? 0 : now->left->SubTreeSize;
            if(index == left_size)break;
            if(index < left_size){
                now = now->left;
            }else{
                now = now->right;
                index -= left_size+1;
            }
        }
        now->splay();
        return now;
    }
 

    /*==============================================================================
        merge/split ベースの実装を行う。
        |- merge(lef,rig) : rig の左の子に lef をつける
        |- split(i,rt) : i 番目のノードの左の辺を無くし、(lef,rig) のペアを返す
        merge と split が反対の操作であることを利用して計算量を削減する
    ==============================================================================*/

    
    // leftroot を根とする SplayTree と、rightroot を根とする SplayTree をこの順に結合する。
    static SplayNode *merge(SplayNode *leftRoot, SplayNode *rightRoot){
        if(leftRoot == nullptr)return rightRoot;// 右のみ
        if(rightRoot == nullptr)return leftRoot;// 左のみ
        assert(leftRoot->parent == nullptr && rightRoot->parent == nullptr);
        // 怖いのでここで update かけときます。大したコストではないです 
        leftRoot->update();
        rightRoot->update();

        rightRoot = get_sub(0,rightRoot);
        rightRoot->left = leftRoot;
        leftRoot->parent = rightRoot;
        rightRoot->update();
        return rightRoot;
    }
    
    // root が根の SplayTree を、左から i-1 番目と i 番目の間で分割する。
    std::pair<SplayNode*,SplayNode*> split(int i, SplayNode*root){
        if(root == nullptr)return std::make_pair(nullptr , nullptr);
        assert(root->parent == nullptr);
        assert(0 <= i && i <= root->SubTreeSize);
        if(i == 0)return std::make_pair(nullptr , root);
        if(i == root->SubTreeSize)return std::make_pair(root, nullptr);
        root = get_sub(i , root);
        auto leftRoot = root->left;
        auto rightRoot = root;
        if(rightRoot != nullptr)rightRoot->left = nullptr;
        if(leftRoot != nullptr)leftRoot->parent = nullptr;
        leftRoot->update();
        rightRoot->update();
        return std::make_pair(leftRoot,rightRoot);
    }
    

    /*
        root が根であるような Splay 木で、以下の条件を満たすノード x の個数を返す。
        lower = true : x < node (lower_bound)
        lower = false: x <= node (upper_bound)
        条件を満たすギリギリの境界を探索する (ただし、境界の左右どちらかは不定)
    */
    static std::pair<SplayNode*,int> bound_sub(SplayNode* Node, SplayNode *root , bool lower , bool key_only_compare){
        if(root == nullptr)return std::make_pair(root,0);
        SplayNode *now = root;// 条件による境界ギリギリのノードを根から探索する
        Node->update();// 念の為集約と遅延評価をしておく
        bool satisfy = false;
        while(true){
            now->eval();
            if(lower){
                // lower_bound (now < Node)
                satisfy = key_only_compare ? bool(now->Key < Node->Key) : bool(*now < *Node);
            }else {
                // upper_bound (now <= Node)
                satisfy = key_only_compare ? !bool(Node->Key < now->Key) : !bool(*Node < *now);
            }
            // now が条件を満たす側にいる場合、右側に境界があるとわかる
            if(satisfy){
                if(now->right == nullptr)break;
                now = now->right;
            }else{
                if(now->left == nullptr)break;
                now = now->left;
            }
        }
        int res = 0;
        // now が条件を満たす側で終了した場合、自身の分を加算する。
        if(satisfy)res = 1;
        now->splay();
        if(now->left != nullptr)res += now->left->SubTreeSize;
        return std::make_pair(now ,res);
    }

    
    // root が根の splaytree に NODE (別の連結成分)を挿入する
    // NODE は update されている必要がある
    SplayNode *insert_sub(SplayNode *NODE , SplayNode *root){
        NODE->update();// 念の為集約と遅延評価をしておく
        if(root == nullptr)return NODE;
        
        // lower_bound で、x < NODE の境界となる x を取得 (ただし境界の右側か左側かは不定)
        root = bound_sub(NODE,root,true,false).first;
        // root が x < NODE の境界の左右どちらかによって、NODE を配置する場所が変わる
        root->eval();
        if(*root < *NODE){ // root < NODE の場合は root の右に
            if(root->right != nullptr)root->right->parent = NODE;
            NODE->right = root->right;
            root->right = nullptr;
            NODE->left = root;
        }else{// NODE <= root の場合は root の左に
            if(root->left != nullptr)root->left->parent = NODE;
            NODE->left = root->left;
            root->left = nullptr;
            NODE->right = root;
        }
        root->parent = NODE;
        root->update();
        NODE->update();
        return NODE;// NODE が根になっている
    }
     

    // upper_bound などで比較を行うためのノードオブジェクト。SplayTree に挿入はしない
    SplayNode* const BluffObject(type_key k , type_value v){
        m_bluff_object->Key = k;
        m_bluff_object->Value = v;
        return m_bluff_object;
    }

    public:

    MyMultiSet(): m_Root(nullptr) , m_paired(true){}
    ~MyMultiSet(){
        // メモリ解放。MyBinaryVector のような要素検索がある場合は、多重解放にならないようにしてね
        while(m_Root != nullptr)this->pop(0);
    }

    // コピー
    MyMultiSet(const MyMultiSet<type_key,type_value>& param_tree) : MyMultiSet() {
        // subr 以降の部分木をコピー
        function<SplayNode*(const SplayNode*)>subr_copy = [&](const SplayNode* subr){
            if(subr == nullptr)return static_cast<SplayNode*>(nullptr);
            SplayNode* res = new SplayNode(*subr);
            res->left = subr_copy(res->left);
            res->right = subr_copy(res->right);
            if(res->left != nullptr)res->left->parent = res;
            if(res->right != nullptr)res->right->parent = res;
            return res;
        };
        m_Root = subr_copy(param_tree.m_Root);
        this->m_paired = param_tree.m_paired;
    }
    
    // ムーブ
    MyMultiSet(MyMultiSet<type_key,type_value>&& param_tree) noexcept : MyMultiSet() {
        swap(this->m_Root , param_tree.m_Root);
        swap(this->m_paired , param_tree.m_paired);
    }
    
    // コピー&スワップ イディオム
    MyMultiSet& operator=(MyMultiSet<type_key,type_value> param_tree){
        swap(this->m_Root , param_tree.m_Root);
        swap(this->m_paired , param_tree.m_paired);
        // param_tree に渡したリソースは param_tree のデストラクタで破棄される
        return *this;
    }

    // 要素数
    int size(){
        if(m_Root == nullptr)return 0;
        return m_Root->SubTreeSize;
    }

    // 左から i 番目のNodeのコピーを取得 (0-index)
    // ただし、値の取得が目的なので right などの隣接頂点へのアクセスを封印する
    SplayNode get(int i){
        assert(0 <= i && i < size());
        m_Root = get_sub(i,m_Root);
        return m_Root->make_temp_instance();
    }


    // [l,r) に対応する部分木の根のコピーを返す。ただし、隣接ノードへのアクセスを封じる。
    // 使用用途はモノイド積の取得に限定 (コピーなので、オリジナルの現在の値とは関係なく [l,r) のデータが格納されている)
    SplayNode GetRange(int l , int r){
        assert(0 <= l && l < r && r <= size());
        auto tmp = split(r,m_Root);
        auto rightRoot = tmp.second;
        tmp = split(l,tmp.first);// 部分木を取り出す。
        auto res = tmp.second->make_temp_instance();
        m_Root = merge(merge(tmp.first,tmp.second),rightRoot);
        return res;
    }
 
    // Keyのみを指定して挿入 ( Value は指定しないので未定義 )
    void insert( type_key key_ ){
        // 未定義の Value が追加されたので、今後 (Key,Value) の pair に注目する操作が行えなくなる
        m_paired = false;
        m_Root = insert_sub(new SplayNode(key_,m_default_val) ,m_Root);
        return;
    }
 
    // KeyとValueを指定して挿入
    void insert_pair( type_key key_ , type_value val_){
        assert(m_paired);
        m_Root = insert_sub(new SplayNode(key_,val_) ,m_Root);
        return;
    }
 
    // index番目のNodeを消去
    void pop(int index){
        assert(0 <= index && index < size());
        auto center = get_sub(index,m_Root);
        auto leftRoot = center->left;
        auto rightRoot = center->right;
        if(leftRoot != nullptr)leftRoot->parent = nullptr;
        if(rightRoot != nullptr)rightRoot->parent = nullptr;
        center->left = nullptr;
        center->right = nullptr;
        center->update();

        m_Root = merge(leftRoot,rightRoot);
        delete center;
    }

    // Keyとして、key_を持つ頂点を一つpop。ただし、Valueについては指定しない
    void erase(type_key key_){
        int it = find(key_);
        if(it!=-1)pop(it);
        return;
    }
 
    /*(key_,val_)を持つ頂点を一つ削除*/
    void erase_pair(type_key key_ , type_value val_){
        assert(m_paired);
        int it = find_pair(key_ , val_);
        if(it!=-1)pop(it);
        return;
    }
 
    // Key が x 未満の頂点の個数を返す    
    int lower_bound(type_key x){
        if(size() == 0)return 0;
        // 比較用 Node オブジェクトを使って比較する
        auto tmp = bound_sub(BluffObject(x,type_value(0)),m_Root,true,true);
        m_Root = tmp.first;
        return tmp.second;
    }

    // (Key , Value)に関して、a < NODE(x,y)であるような頂点の個数を返す
    int lower_bound_pair(type_key x , type_value y){
        assert(m_paired);
        if(size() == 0)return 0;
        auto tmp = bound_sub(BluffObject(x,y),m_Root,true,false);
        m_Root = tmp.first;
        return tmp.second;
    }
 
    // Key が x 以下の頂点の個数を返す
    int upper_bound(type_key x){
        if(size() == 0)return 0;
        auto tmp = bound_sub(BluffObject(x,type_value(0)),m_Root,false,true);
        m_Root = tmp.first;
        return tmp.second;
    }
 
    // (Key , Value)に関して、a <= NODE(x,y)であるような頂点の個数を返す
    int upper_bound_pair(type_key x , type_value y){
        assert(m_paired);
        if(size() == 0)return 0;
        auto tmp = bound_sub(BluffObject(x,y),m_Root,false,false);
        m_Root = tmp.first;
        return tmp.second;
    }
    

    // x の個数を返す
    int count(type_key x){
        return upper_bound(x) - lower_bound(x);
    }

    // (x,y) の個数を返す
    int count_pair(type_key x , type_key y){
        assert(m_paired);
        return upper_bound_pair(x,y) - lower_bound_pair(x,y);
    }

    
    // Key == x である index (複数あるなら最初のもの) を返す (0-index)。なければ-1を返す
    int find(type_key x){
        if(size() == 0)return -1;
        if(count(x) == 0)return -1;
        return lower_bound(x);
    }

    // (Key,Value) == (x,y) である index (複数あるなら最初のもの) を返す (0-index)。なければ-1を返す
    int find_pair(type_key x , type_value y){
        assert(m_paired);
        if(size() == 0)return -1;
        if(count_pair(x,y) == 0)return -1;
        return lower_bound_pair(x,y);
    }

    SplayNode back(){
        assert(size()>0);
        return get(size()-1);
    }

    SplayNode front(){
        assert(size()>0);
        return get(0);
    }

    void pop_back(){
        assert(size()>0);
        pop(size()-1);
    }
    // 先頭を削除
    void pop_front(){
        assert(size()>0);
        pop(0);
    }
