はじめに

Euler Tour Tree (以降、ETT とします) は根付きの 動的木 ないしは 動的森 を管理するデータ構造です。この記事を読む前に平衡二分木 と Link-Cut Tree について学んでいると良いです。


図 1

Link-Cut Tree もまた動的森を管理するデータ構造ですが、Link-Cut Tree ではパスに関するデータ取得がメインでした。ETT では、木の DFS 順のデータを取得できます。

また、Link-Cut Tree と組み合わせることで木の 部分木 に対するデータ取得も可能になります。


データ構造


概要

オイラーツアー列を以下で定義します。

  • 木の根からの DFS を行い、通った辺と頂点を記録した列。
  • 辺は通った順に記録する。
  • 頂点を通るタイミングは何度か存在するので、そのうちいずれか一度のみを記録する。

ETT は動的森の各連結成分についてオイラーツアー列を管理するデータ構造です。


図 2

例えば図 2 は各連結成分の DFS の道順の \(1\) つを示しています。この道順に対して、オイラーツアー列のひとつを示したのが以下の図です。


図 3

ここで、改めてオイラーツアー列の構造について補足しておきます。図において、丸枠の要素が頂点に対応しており、長方形の要素が辺に対応しています。便宜上、頂点に対応する要素を頂点要素、辺に対応する要素を辺要素と呼ぶことにします。

図 3 においては、頂点要素は初めて頂点に到達するのと同じタイミングでオイラーツアー列に登場していますが、これはたまたまそうなっているだけです。ここでのオイラーツアー列の定義では、頂点要素の登場箇所については強く保証されていないことに注意しておいてください。

オイラーツアー列は順序を持つので、これを平衡二分木で管理することにします。


図 4

ETT では、森ないし森における操作をここで定義したオイラーツアー列で表現することで、DFS 順のデータを管理します。

また実装においては、平衡二分木とは別でオイラーツアー列の要素を指すポインタを以下のデータ構造で管理しておきます。

  • \(\mathrm{NodeFactor}[x] := \) 頂点 \(x\) に対応する頂点要素を指すポインタ
  • \(\mathrm{EdgeFactor}[u][v] := \) 頂点 \(u\) から頂点 \(v\) への有効辺に対応する頂点要素を指すポインタ

これらは 配列 と ハッシュテーブル を組み合わせることで十分高速に管理できます。


連結成分の根の変更

連結成分の根の変更メソッドとして、以下を定義します。

  • \(\mathrm{evert}(x) := \) 頂点 \(x\) が属する連結成分の根を \(x\) に変更する。

図 5

連結成分の根を変更する手続きは、根にしたい頂点が属する連結成分のオイラーツアー列の順番を入れ替えることで実行可能です。具体的には、\(\mathrm{evert}(x)\) では以下の手続きを行います。

  1. 頂点 \(x\) に対応する頂点要素が、オイラーツアー列において何番目の要素かを特定する。これは平衡二分木の機能で実行可能。
  2. 頂点 \(x\) に対応する頂点要素の直前でオイラーツアー列を二分し、それらを元と反対の順で結合する。

以下の図は、頂点 \(6\) が属する連結成分の根を頂点 \(6\) にする操作の様子です。


図 6

得られた列もちゃんと頂点 \(6\) からのオイラーツアー列のひとつであることが確認できます。


辺の追加

動的森なので、辺の追加を行うメソッドを以下で定義します。

  • \(\mathrm{link}(u,v) := \) 頂点 \(u\) と頂点 \(v\) を辺で繋ぐ。
    • \(u\) と \(v\) が同じ連結成分に属している状態で呼び出すとエラーとする。
    • 連結成分をマージした後、その連結成分の根は頂点 \(u\) であるとする。

図 7

手続きは以下の \(2\) ステップからなります。

  1. 頂点 \(u\) と \(v\) を \(\mathrm{evert}\) する。
  2. 以下を順に連結してひとつのオイラーツアー列にする。
    1. 頂点 \(u\) からのオイラーツアー列
    2. 頂点 \(u\) から頂点 \(v\) への辺要素 (新たに生成)
    3. 頂点 \(v\) からのオイラーツアー列
    4. 頂点 \(v\) から頂点 \(u\) への辺要素 (新たに生成)

図 8

図を確認すると、頂点 \(u\) を根とするオイラーツアー列になっていることが確認できます。


辺の削除

最後に、辺の削除を行うメソッドを以下で定義します。

  • \(\mathrm{cut}(u,v) := \) 頂点 \(u\) と頂点 \(v\) を繋ぐ辺を削除する。
    • 頂点 \(u\) と頂点 \(v\) が辺で繋がれていない状態で呼び出すとエラーとする。

図 9

手続きは以下の \(3\) ステップからなります。

  1. 頂点 \(u\) を \(\mathrm{evert}\) する。
  2. 頂点 \(u\) からのオイラーツアー列を、\(2\) つの辺要素 \((u \rightarrow v)\) , \((v \rightarrow u)\) の登場地点で \(\mathrm{split}\) し、辺要素 \((u \rightarrow v)\) , \((v \rightarrow u)\) は削除する。
  3. 手続き \(2\) の \(\mathrm{split}\) で得た \(3\) つの列を左から順に \(A,B,C\) とし、頂点 \(u\) からのオイラーツアー列を \(A\) と \(C\) をこの順に連結したもの、頂点 \(v\) からのオイラーツアー列を \(B\) とする。

図 10

手続きの図を確認すると、それぞれオイラーツアー列になっていることが確認できます。


図 11

データの更新/取得


根の取得

頂点 \(x\) が属する連結成分の根はオイラーツアーの開始点なので、頂点 \(x\) からのオイラーツアー列の最左の要素をみることで特定することができます。頂点 \(x\) に対応する頂点要素のポインタを \(\mathrm{NodeFactor}[x]\) で管理しているので、その頂点から平衡二分木の根まで移動し、その後左に降り続けることで列の最左に到達することができます。

また、二つの頂点に対して、連結成分の根が一致するかをみることで、同じ連結成分に属するかの判定も可能です。


部分木

オイラーツアー列は平衡二分木で表現されているので、区間に対して集約の計算や遅延評価による更新を行うことができます。

オイラーツアー列からうまく区間を選択すると、元の木における部分木と対応する区間を得ることができます。もし頂点 \(x\) の親頂点 \(p\) がわかっているなら、\(2\) つの辺要素 \((p\rightarrow x)\) と \((x\rightarrow p)\) の間の要素を選ぶことで、頂点 \(x\) の部分木に対応する区間を得ることができます。

ETT で親頂点を特定することは難しいので、Link-Cut Tree で親を特定してから操作を行うと良いです。なお、親がない場合、すなわち \(x\) が根頂点の場合、部分木は木全体のことなので、オイラーツアー全体を見れば良いです。


図 12

オイラーツアーの頂点要素には頂点重みを、辺要素には辺重みを持たせることにしましょう。先述で得た区間内に登場する頂点要素も辺要素も、頂点 \(x\) の部分木に含まれるものなので、この区間に対して区間の操作を行うことで部分木に対する操作を行うことができます。

実際に区間操作を行う際、頂点要素と辺要素を分けて考える必要があることに注意してください。具体的には、頂点要素に対する集約の計算や遅延評価の際は辺要素を見ないようにし、辺要素に対する集約の計算や遅延評価の際は頂点要素を見ないように実装してください。

また、Splay Tree の部分木内の頂点要素の個数など、Link-Cut Tree などでは使わないような集約も必要な場面があるので注意してください。

/*
    Copyright © (c) okoteiyu 2024. 
    Released under the MIT license(https://opensource.org/licenses/mit-license.php) 

    オイラーツアー森。木を明示的に管理するのではなく、木のオイラーツアーを SplayTree (列型) として管理する。

    - オイラーツアー列は以下の 2 要素からなる
        |- u から v に移動したことを表す EdgeFactor (u → v) 
        |- 頂点 u を表す NodeFactor (u → u)
            |- NodeFactor(u → u) が u の部分木に対応する区間のどこに現れるかは不定
    - これらはオイラーツアーに重複して現れることはない
    |
    - NodeFactor も EdgeFactor も SplayNode で表現する。
        |- 頂点、辺のどちらか一方に値を持たせることができるように設計
        |- 値を持たせる方を product_target ということにする。
            |- テンプレートパラメータ edge_focused が true なら頂点重み
            |- false なら辺重み
        |- product_target ではない方の集計は無視する。
    |
    - 森としてのメソッド
        |- root(x) := x の属する木の根
        |- same(x,y) := x,y が同じ木に属しているか
        |- ComponentSize(r) := r が属する木のサイズ
        |- EulerTourSize(r) := r が属する木における r からのオイラーツアーのサイズ
        |- evert(x) := x を連結成分の root とする。副作用として、x の頂点要素が splay される。
        |- exist_edge(x,y) := x,y 間に辺が存在するかどうか
        |- link(x,y) := x,y に辺を追加 (x,y がすでに同じ連結成分内ならエラー)
            |- 辺が値を持つなら、コンストラクタで持たせた init_v を持つ
            |- x を evert
        |- link(x,y,w) := x,y に、値 w を持つ辺を追加 (x,y がすでに同じ連結成分内ならエラー)
            |- x を evert
        |- cut(x,y) := x,y から辺を削除 (x,y 間に辺がなければエラー)
            |- x,y を evert
    |
    - 頂点 u の部分木に関するクエリを処理できるが、頂点 u の親頂点 p がわかっていないといけない。
        |- 頂点 u の部分木に対応するオイラーツアーの区間を見つけるため (他にうまくやる方法はない)
        |- u の親は ETT から計算することはできないので、LinkCutTree と併用する。
        |- u が根の場合は親を考えなくて OK (u が根なら p はなんでも (テキトーで) OK)
    |
    - operator [u] := 頂点 u の m_NodeFactor の SplayNode を返す
    - SubTree(u,p) := 「u の部分木に対応するオイラーツアー列の区間」に対応する SplayNode のコピーを取得 (p は u の親)
        |- product_target が持つ値についてのモノイド積 (sum,max など) はここから取得する
    - (注意) SplayNode のコピーは、隣接する SplayNode へのポインタを封印して返す
    | 
    - update_val(i,x) := 頂点 i の持つ値を x にする。product_target が辺の時はエラー
    - update_val(u,v,x) := 辺(u-v) の持つ値を x にする。product_target が頂点の時はエラー
    - SubTreeUpdate(u,p,x) := u の部分木 (u 含む) の product_target の持つ値を一律に x に変更する  (p は u の親)
    - SubTreeAffine(u,p,a,b) := u の部分木 (u 含む) の product_target の持つ値に一律に a かけて b 足す (p は u の親)
    - SubTreeAdd(u,p,x) := u の部分木 (u 含む) の product_target の持つ値に一律に x 足す (p は u の親)
    |
    - tour(i,r) := r が属する木における r からのオイラーツアーの i 番目の SplayNode (のコピー) を返す
    - euler_tour := オイラーツアー到達順に頂点を並べた vector
    - euler_tour_compress := オイラーツアーで、部分木に入る時/出る時 のみ記録
    |
    - product_target の持つ値 (Value) は T 型。
        |- 頂点、辺のどちらか一方に値を持たせることができるように設計
        |- テンプレートパラメータ edge_focused が false なら頂点重み。true なら辺重み
    - 遅ければいらない遅延評価やモノイド積を消す。
        |- SubTreeSize は必須
        |- NodeFactor,EdgeFactor が混在していて集約が若干ややこしいので注意が必要
*/
template<typename T , bool edge_focused = false >
class EulerTourTree{
    /*   
        Copyright © (c) okoteiyu 2024. 
        Released under the MIT license(https://opensource.org/licenses/mit-license.php) 
    */

    private:
    struct SplayNode{
        friend class EulerTourTree;

        public:
        
        // オイラーツアーの移動方向の両端点
        long long from , to;

        T Value;//値 (辺か頂点のいずれかに割り振られる)
        T Min,Max,Sum;//部分木のうち、値の最大、最小、和
        long long  SubTreeSize = 1;

        private:
        /* SplayNode としてのメンバ */
        SplayNode *parent = nullptr;//親ノード
        SplayNode *left = nullptr;//左の子ノード
        SplayNode *right = nullptr;//右の子ノード

        inline static const pair<T,T> eq_affine = {T(1),T(0)}; // アフィン変換の単位元
        pair<T,T> lazy_affine = eq_affine;// 遅延中のアフィン変換のパラメータ

        // 部分木のうち、頂点要素の個数
        long long SubTreeNodeFactorCount = 0;

        // SplayTree としての部分木含まれる集約のターゲット (辺or頂点) の個数
        long long TargetFactorCount = 0;

        /*==================================================
            メモリアライメントに配慮して、bool は最後に宣言する
        ===================================================*/

        // ノードのコピーかどうか 
        bool temp_instance = false;
        // 集約の対象かどうか (辺は片方向にのみ値を載せたい)
        bool is_product_target;
        

        SplayNode(){}
        SplayNode(long long from_, long long to_, T val , bool is_product_target_)
            :from(from_),to(to_),Value(val),is_product_target(is_product_target_)
        {
            this->update();    
        }

        // デフォルトのシャローコピーを採用
        SplayNode(const SplayNode&) = default;
        SplayNode(SplayNode&&) = default;
        SplayNode& operator=(const SplayNode&) = default;
        SplayNode& operator=(SplayNode&&) = default;

        // 隣接頂点へのアクセスを禁止したコピーを返す。
        // コピーに対する splay や update などは invalid
        SplayNode make_temp_instance(){
            assert(temp_instance == false);
            SplayNode res = *this;
            res.left = nullptr;
            res.right = nullptr;
            res.parent = nullptr;
            res.temp_instance = true;
            return res;
        }


        // アフィン変換を遅延させる
        inline void set_lazyAffine(T a, T b) noexcept {
            this->lazy_affine = {a*this->lazy_affine.first , a*this->lazy_affine.second + b};
        }
        


        // this - this->parent を回転させる (親は必ず存在するものとし、位置関係も誤りがないものとする)
        void rotate(){
            if(this->parent->parent != nullptr){//GrandParentの子を張り替え
                if(this->parent == this->parent->parent->left)this->parent->parent->left = this;
                else this->parent->parent->right = this; 
            }
            this->parent->eval();
            this->eval();
            //元の親に対する位置で場合分け
            if(this->parent->left == this){
                this->parent->left = this->right;
                if(this->right != nullptr)this->right->parent = this->parent;
                this->right = this->parent;
                this->parent = this->right->parent;
                this->right->parent = this;
                this->right->update();
            }else{
                this->parent->right = this->left;
                if(this->left != nullptr)this->left->parent = this->parent;
                this->left = this->parent;
                this->parent = this->left->parent;
                this->left->parent = this;
                this->left->update();
            }
            this->update();
            return;
        }
        /*
            自分から見た親の位置
            0 -> 親が存在しない
            1 -> 親の左の場合
            2 -> 親の右の場合
        */
        int state() noexcept {
            if(this->parent == nullptr) [[unlikely]] return 0;
            this->parent->eval();
            if(this->parent->left == this)return 1;
            return 2;
        }

        // あるNodeを回転を駆使し一番上まで持っていく
        void splay() noexcept {
            while(this->parent != nullptr){
                if(this->parent->state() == 0){
                    this->rotate();
                    break;
                }
                // 親の state をすでに計算しているので、親の親 → 親 → 自分に reverse の評価を降ろせている
                if( this->parent->state() == this->state() )this->parent->rotate();
                else this->rotate();
                this->rotate();
            }
            this->update();// while が呼ばれなかった場合のため
            return;
        }


        // Nodeの持っている情報をupdateする
        void update() noexcept {
            assert(temp_instance == false);
            
            this->eval();
            this->SubTreeSize = 1;

            this->TargetFactorCount = 0;
            if(this->is_product_target)this->TargetFactorCount++;

            // get_tour 用メンバ
            SubTreeNodeFactorCount = 0;
            if(this->from == this->to)SubTreeNodeFactorCount = 1;

            // product_target がない区間は集約せず、値も未定義のままで良い
            this->Sum = this->Value;
            this->Min = this->Value;
            this->Max = this->Value;

            // 先に、辺/頂点 共通のメンバを集約
            if(this->left != nullptr){
                this->left->eval();
                this->SubTreeSize += this->left->SubTreeSize;
                this->TargetFactorCount += this->left->TargetFactorCount;
                this->SubTreeNodeFactorCount += this->left->SubTreeNodeFactorCount;
                // 集約対象をもつ子からのみ集約
                if(this->left->TargetFactorCount > 0){
                    this->Min = min<T>(this->left->Min,this->Min);
                    this->Max = max<T>(this->left->Max,this->Max);
                    this->Sum += this->left->Sum;
                }
            }
            if(this->right != nullptr){
                this->right->eval();
                this->SubTreeSize += this->right->SubTreeSize;
                this->TargetFactorCount += this->right->TargetFactorCount;
                this->SubTreeNodeFactorCount += this->right->SubTreeNodeFactorCount;
                if(this->right->TargetFactorCount > 0){
                    this->Min = min<T>(this->right->Min,this->Min);
                    this->Max = max<T>(this->right->Max,this->Max);
                    this->Sum += this->right->Sum;
                }
            }
        }
        

        /* この頂点にアクセスされるたびに、各flagがtrueなら遅延させていた処理を実行し、左右の子にflagを伝播する */
        void eval(){
            assert(temp_instance == false);
            
            // product_target がない区間は遅延評価をかけず、Sum などはモノイドの単位元を維持する 
            if(this->TargetFactorCount == 0){
                this->lazy_affine.first = false;// 遅延評価を解消するのを忘れずに
                return;
            }

            // アフィン変換
            if(this->lazy_affine == eq_affine){
                // 値をもつ対象のみ更新
                if(this->is_product_target)this->Value = this->lazy_affine.first * this->Value + this->lazy_affine.second;

                this->Min = this->lazy_affine.first * this->Min + this->lazy_affine.second;
                this->Max = this->lazy_affine.first * this->Max + this->lazy_affine.second;
                this->Sum = this->lazy_affine.first * this->Sum + this->TargetFactorCount*this->lazy_affine.second;
                if(this->Max < this->Min)swap(this->Max,this->Min);
                
                if(this->left != nullptr)this->left->set_lazyAffine(this->lazy_affine.first,this->lazy_affine.second);
                if(this->right != nullptr)this->right->set_lazyAffine(this->lazy_affine.first,this->lazy_affine.second);
                this->lazy_affine = eq_affine;//伝播したらフラグを折る
            }
        }
    };

    protected:
    /*=================================
                メンバ変数たち
    =================================*/

    // 頂点数 (頂点番号は [0,m_N-1] )
    int m_N;
    // unordered_map は遅いのでなるべくアクセス回数を減らす
    // [x] := 頂点 x を表す特殊 SplayNode (x→x で、EulerTour に一度のみ登場する )
    vector<SplayNode*> m_NodeFactor;
    // [u][v] := u → v の移動を表す SplayNode 
    vector<unordered_map<long long,SplayNode*>> m_EdgeFactor;


    private:
   
    // rootを根とする部分木において、求めたいindexまで降りていく(左側の部分木サイズを参照する)
    static SplayNode *get_sub(int index, SplayNode* root){
        if(root == nullptr)return root;
        assert(root->parent == nullptr);

        auto now = root;
        while(true){
            now->eval();// ノードを見る前にeval
            int left_size = bool(now->left == nullptr) ? 0 : now->left->SubTreeSize;
            if(index == left_size)break;
            if(index < left_size){
                now = now->left;
            }else{
                now = now->right;
                index -= left_size+1;
            }
        }
        now->splay();
        return now;
    }

    /*==============================================================================
        merge/split ベースの実装を行う。
        |- merge(lef,rig) : rig の左の子に lef をつける
        |- split(i,rt) : i 番目のノードの左の辺を無くし、(lef,rig) のペアを返す
        merge と split が反対の操作であることを利用して計算量を削減する
    ==============================================================================*/

    
    // leftroot を根とする SplayTree と、rightroot を根とする SplayTree をこの順に結合する。
    static SplayNode *merge(SplayNode *leftRoot, SplayNode *rightRoot){
        if(leftRoot == nullptr)return rightRoot;// 右のみ
        if(rightRoot == nullptr)return leftRoot;// 左のみ
        assert(leftRoot->parent == nullptr && rightRoot->parent == nullptr);
        // 怖いのでここで update かけときます。大したコストではないです 
        leftRoot->update();
        rightRoot->update();

        rightRoot = get_sub(0,rightRoot);
        rightRoot->left = leftRoot;
        leftRoot->parent = rightRoot;
        rightRoot->update();
        return rightRoot;
    }
    
    // root が根の SplayTree を、左から i-1 番目と i 番目の間で分割する。
    std::pair<SplayNode*,SplayNode*> split(int i, SplayNode*root){
        if(root == nullptr)return std::make_pair(nullptr , nullptr);
        assert(root->parent == nullptr);
        assert(0 <= i && i <= root->SubTreeSize);
        if(i == 0)return std::make_pair(nullptr , root);
        if(i == root->SubTreeSize)return std::make_pair(root, nullptr);
        root = get_sub(i , root);
        auto leftRoot = root->left;
        auto rightRoot = root;
        if(rightRoot != nullptr)rightRoot->left = nullptr;
        if(leftRoot != nullptr)leftRoot->parent = nullptr;
        leftRoot->update();
        rightRoot->update();
        return std::make_pair(leftRoot,rightRoot);
    }


    // u が属する木のオイラーツアーでの、u の m_NodeFactor の出現位置を答える (0-index)
    // 副作用 : u の m_NodeFactor が splay される
    long long find_NodeFactor(long long u){
        assert(0 <= u && u < m_N);
        auto u_nd = m_NodeFactor[u];
        u_nd->splay();
        if(u_nd->left != nullptr)return u_nd->left->SubTreeSize;
        return 0;
    }

    // u,v が属する木のオイラーツアーでの、u→v と移動する m_EdgeFactor の出現位置を答える (0-index)
    // なければ -1
    // 副作用 : u の m_NodeFactor が splay される
    long long find_EdgeFactor(long long u , long long v){
        assert(0 <= u && u < m_N);
        assert(0 <= v && v < m_N);
        auto uv_edge = m_EdgeFactor[u][v];
        if(uv_edge == nullptr)return -1;
        uv_edge->splay();
        if(uv_edge->left != nullptr)return uv_edge->left->SubTreeSize;
        return 0;
    }


    // 頂点 u 以下の部分木のオイラーツアーに対応する区間の SplayNode* (SpalyTree の根) を切り離す。
    // {左側 の SplayNode , 真ん中の SplayNode, 右側の SplayNode }}
    // 必ず戻す必要がある (辺を cut するわけではない)
    // 副作用 : 要件なし(未定義)
    tuple<SplayNode*,SplayNode*,SplayNode*> SplitSubTree(long long u , long long p){
        assert(0 <= u && u < m_N);
        assert(0 <= p && p < m_N);
        long long rt = root(u);// 木の根
        auto u_nd = m_NodeFactor[u];
        // u が木の根の場合
        if(u == rt){
            u_nd->splay();
            return {nullptr,u_nd,nullptr};
        }
        assert(m_EdgeFactor[p][u] != nullptr);// 辺が存在するか assert
        evert(p);
        // u の部分木のオイラーツアー区間を切り取る (入る辺 , 出る辺も含む)
        long long l = find_EdgeFactor(p,u);
        long long r = find_EdgeFactor(u,p);
        assert(l < r);// p の情報が正しいか assert
        u_nd->splay();
        pair<SplayNode*,SplayNode*> tmp = split(r,u_nd);
        SplayNode* rightnode = tmp.second;
        tmp = split(l+1,tmp.first);
        return {tmp.first, tmp.second, rightnode};
    }


    public:

    EulerTourTree(){}
    // 頂点数と product_target の値の初期値を指定
    EulerTourTree(int n ,T init_v = T()) : m_N(n){
        m_NodeFactor.clear();
        m_EdgeFactor.clear();
        for(int i = 0 ; i < m_N ; i++){
            m_NodeFactor.push_back(new SplayNode(i,i,init_v,!edge_focused));
        }
        m_EdgeFactor.resize(m_N);
    }
    // デストラクタ
    ~EulerTourTree(){
        for(SplayNode* p : m_NodeFactor){
            if(p == nullptr)continue;
            delete p;
        }
        for(unordered_map<long long,SplayNode*> table : m_EdgeFactor){
            for(pair<long long,SplayNode*>e : table){
                if(e.second == nullptr)continue;
                delete e.second;
            }
        }
    }

    // 複雑な挙動を回避するので、コンストラクタによるコピー/ムーブを一律に禁止する。
    EulerTourTree(const EulerTourTree<T>&) = delete ;
    EulerTourTree<T>& operator=(const EulerTourTree<T>&) = delete ;
    EulerTourTree(EulerTourTree<T>&&) = delete;
    EulerTourTree<T>& operator=( EulerTourTree<T>&& ) = delete;

    // 頂点 u,v 間に辺が存在するかどうか
    bool exist_edge(long long u, long long v){
        assert(0 <= u && u < m_N);
        assert(0 <= v && v < m_N);
        return bool(m_EdgeFactor[u][v] != nullptr && m_EdgeFactor[v][u] != nullptr);
    }



    // 頂点 i (の NodeFactor) のもつ値(value)を x に変更する
    // 副作用 : i の m_NodeFactor を splay
    void update_val(long long i , T x){
        assert(0 <= i && i < m_N);
        assert(!edge_focused);// 頂点重みの場合のみ
        auto i_nd = m_NodeFactor[i];
        i_nd->splay();
        assert(i_nd->is_product_target);
        i_nd->Value = x;
        i_nd->update();
    }

    // 辺 (u-v) の値を w にする
    // 値は 2 つの辺要素の片方に持たせてある。
    // 副作用 : 値を持つ方を splay する。
    void update_val(long long u , long long v , T w){
        assert(0 <= u && u < m_N);
        assert(0 <= v && v < m_N);
        assert(edge_focused);// 辺重みの場合のみ        
        assert(exist_edge(u,v));
        SplayNode* e1 = m_EdgeFactor[u][v];
        SplayNode* e2 = m_EdgeFactor[v][u];
        if(e2->is_product_target)swap(e1,e2);
        assert(e1->is_product_target);
        e1->splay();
        e1->Value = w;
        e1->update();
    }

    // r が属する木における r からのオイラーツアーのサイズ
    // 属する木のサイズ個の m_NodeFactor と、(サイズ-1)×2 個の m_EdgeFactor からなる
    // 副作用 : evert(r)
    long long EulerTourSize(long long r){
        assert(0 <= r && r < m_N);
        evert(r);// evert には splay の副作用がある
        return m_NodeFactor[r]->SubTreeSize;
    }
    // u が属する木のサイズ
    // 副作用 : u の m_NodeFactor が splay される
    long long ComponentSize(long long u){
        assert(0 <= u && u < m_N);
        return (EulerTourSize(u) + 2)/3;// 復元する
    }

    // 頂点 u が属する木の根の番号
    // 副作用 : 要件なし
    long long root(long long u){
        assert(0 <= u && u < m_N);
        SplayNode* tmp = m_NodeFactor[u];
        tmp->splay();
        tmp->eval();
        while(tmp->left != nullptr){
            tmp = tmp->left;
            tmp->eval();
        }
        tmp->splay();
        return tmp->from;
    }

    // u,v が同じ連結成分に属するかどうか
    // 副作用 : u,v の m_NodeFactor が未定義の順に splay される
    // SplayTree の根を見て判定する
    bool same(long long u ,long long v){
        auto u_nd = m_NodeFactor[u];
        auto v_nd = m_NodeFactor[v];
        auto u_now = u_nd;
        auto v_now = v_nd;

        while(u_now->parent != nullptr)u_now = u_now->parent;
        while(v_now->parent != nullptr)v_now = v_now->parent;

        u_nd->splay();
        v_nd->splay();

        return bool(u_now == v_now);
    }

    // 頂点 u を、属する木の根とする。
    // evert(u) すると、u の m_NodeFactor (u→u) が オイラーツアーの先頭になる
    // 副作用 : (u→u) が splay される (必須要件)
    void evert(long long u){
        assert(0 <= u && u < m_N);
        long long c = find_NodeFactor(u);
        auto u_nd = m_NodeFactor[u];
        u_nd->splay();
        if(u_nd->left == nullptr)return;// すでに根である
        pair<SplayNode*,SplayNode*> tmp = split(c, u_nd);
        merge(tmp.second,tmp.first);
        u_nd->splay();
    }


    // u-v をリンクする (u の子に v をつける)
    // 副作用 : u の m_NodeFactor が splay される。u が evert される
    void link(long long u , long long v){
        assert(!edge_focused);// 辺集約があるなら、必ず重みを指定する。
        assert(!same(u,v));
        assert(0 <= u && u < m_N);
        assert(0 <= v && v < m_N);
        
        auto new_uv = new SplayNode(u,v,T(),false);
        auto new_vu = new SplayNode(v,u,T(),false);
        m_EdgeFactor[u][v] = new_uv;
        m_EdgeFactor[v][u] = new_vu;
        evert(u);
        evert(v);
        merge(merge(merge(m_NodeFactor[u],new_uv),m_NodeFactor[v]),new_vu);
    }


    // u-v を、値として w 持つ辺でリンクする (u の子に v をつける)
    // 副作用 : u が evert される
    void link(long long u , long long v , T w){
        assert(!same(u,v));
        assert(0 <= u && u < m_N);
        assert(0 <= v && v < m_N);
        assert(edge_focused);
        
        auto new_uv = new SplayNode(u,v,w,true);
        auto new_vu = new SplayNode(v,u,w,false);
        m_EdgeFactor[u][v] = new_uv;
        m_EdgeFactor[v][u] = new_vu;
        evert(u);
        evert(v);
        merge(merge(merge(m_NodeFactor[u],new_uv),m_NodeFactor[v]),new_vu);
    }



    // u-v をカットする。(辺が存在している前提)
    // 副作用 : u (,v) が evert される
    void cut(long long u , long long v){
        assert(0 <= u && u < m_N);
        assert(0 <= v && v < m_N);
        assert(exist_edge(u,v));
        evert(u);
        // v の部分木に対応する区間を切り取る
        long long l = find_EdgeFactor(u,v);
        long long r = find_EdgeFactor(v,u);
        auto u_nd = m_NodeFactor[u];
        u_nd->splay();
        std::pair<SplayNode*,SplayNode*> tmp = split(r+1, u_nd);
        tmp.first = split(r,tmp.first).first;// (v→u) の m_EdgeFactor を切り取る
        tmp.first = split(l+1,tmp.first).first;// v を根とする部分を切り取る
        tmp.first = split(l,tmp.first).first;// (u→v) の m_EdgeFactor を切り取る
        // u の部分をマージする
        merge(tmp.first, tmp.second);
        // 切った辺は解放
        delete m_EdgeFactor[u][v];
        delete m_EdgeFactor[v][u];
        m_EdgeFactor[u][v] = nullptr;
        m_EdgeFactor[v][u] = nullptr;
    }
    
    // 頂点 u の部分木の product_target 全ての Value を A*Value + B にする
    // u の親 p を引数で与えないといけない (lct と併用すると ok)
    // ただし、u が根の場合は p はなんでも OK 
    void SubTreeAffine(long long u , T A , T B , long long p){
        assert(0 <= u && u < m_N);
        auto tmp = SplitSubTree(u,p);
        get<1>(tmp)->set_lazyAffine(A,B);
        get<1>(tmp)->update();
        merge(merge(get<0>(tmp),get<1>(tmp)),get<2>(tmp));
    }
    // 頂点 u の部分木の product_target 全ての Value を Value + x にする
    // u の親 p を引数で与えないといけない (lct と併用すると ok)
    // ただし、u が根の場合は p はなんでも OK 
    void SubTreeAdd(long long u , T x , long long p ){
        assert(0 <= u && u < m_N);
        SubTreeAffine(u,T(1),x,p);
    }

    // 頂点 u の部分木の product_target 全ての Value を x にする
    // u の親 p を引数で与えないといけない (lct と併用すると ok)
    // ただし、u が根の場合は p はなんでも OK 
    void SubTreeUpdate(long long u , T x , long long p){
        assert(0 <= u && u < m_N);
        SubTreeAffine(u,T(0),x,p);
    }

    // 頂点 u の部分木に対応するオイラーツアーの区間を表す SplayNode のコピーを取得
    // u の親 p を引数で与えないといけない (lct と併用すると ok)
    // ただし、u が根の場合は p はなんでも OK 
    // read-only なので SplayNode の隣接ノードへのポインタを封印したものを返す
    SplayNode SubTree(long long u , long long p){ 
        assert(0 <= u && u < m_N);
        auto tmp = SplitSubTree(u,p);    
        // コピーを返す    
        auto res = get<1>(tmp)->make_temp_instance();
        merge(merge(get<0>(tmp),get<1>(tmp)),get<2>(tmp));

        return res;
    }


    // r が属する木で、r を根としたオイラーツアーの i 番目の要素 (i は 0-index )
    // 副作用 : r の属する木の根を r にする
    SplayNode tour(int i , long long r){
        assert(0 <= r && r < m_N);
        assert(0 <= i && i < this->EulerTourSize(r));
        evert(r);
        SplayNode* rt = m_NodeFactor[r];
        rt = get_sub(i,rt);
        SplayNode res = rt->make_temp_instance();
        return res;
    }

    // r が属する木で、r を根としたオイラーツアーをデバッグ出力
    // 副作用 : r の属する木の根を r にする
    void Debug(long long r){
        assert(0 <= r && r < m_N);
        evert(r);
        long long sz = EulerTourSize(r);
        cerr << "Debug From Root : " << root(r) << endl;
        SplayNode* rt = m_NodeFactor[r];
        for( int i = 0 ; i < sz ; i++){
            rt = get_sub(i,rt);
            cerr << "( " << rt->from << " , " << rt->to << " ) , value = " << rt->Value << endl; 
        }
        m_NodeFactor[r]->splay();
    }

    // r が属する木で、r を根としたオイラーツアーで index 回目に辿る辺要素 (0-index)
    SplayNode* get_tour(long long r, long long index){
        assert(0 <= r && r < m_N);
        SplayNode* now = m_NodeFactor[r];
        now->splay();
        assert(0 <= index && index < now->SubTreeSize - now->SubTreeNodeFactorCount);
        
        while(true){
            now->eval();// ノードを見る前にeval
            int left_size = 0;
            if(now->left != nullptr){
                left_size = now->left->SubTreeSize - now->left->SubTreeNodeFactorCount;
            }

            if(index == left_size)break;
            if(index < left_size){
                now = now->left;
            }else{
                now = now->right;
                index -= left_size+1;
            }
        }
        now->splay();
        return now->make_temp_instance();
    }

    // r が属する木で、r を根としたオイラーツアー (頂点への到達を全て記録)
    // 長さは辺数*2 + 1 
    vector<long long> euler_tour(long long r){
        assert(0 <= r && r < m_N);
        evert(r);
        vector<long long> res(0);
        SplayNode* rt = m_NodeFactor[r];
        long long sz = rt->SubTreeSize;
        rt = get_sub(0,rt);
        res.push_back(rt->to);
        for( int i = 0 ; i < sz ; i++){
            rt = get_sub(i,rt);
            if(rt->from != rt->to)res.push_back(rt->to);
        }
        return res;
    }

    // r が属する木で、r を根としたオイラーツアー の圧縮 ver.
    // 部分木に入る時と出る時のみをメモする
    vector<long long> euler_tour_compress(long long r){
        assert(0 <= r && r < m_N);
        evert(r);
        vector<long long> res,tmp;
        tmp = this->euler_tour(r);
        // [x] := tour で x が最初に登場する地点 
        unordered_map<long long,priority_queue<long long>> first;
        // [x] := tour で x が最後に登場する地点 
        unordered_map<long long,priority_queue<long long>> last;
        for(int i =0 ; i < tmp.size() ; i++){
            first[tmp[i]].push(-i);
            last[tmp[i]].push(i);
        }
        for(int i =0 ; i < tmp.size() ; i++){
            if(i == abs(first[tmp[i]].top()))res.push_back(tmp[i]);
            if(i == abs(last[tmp[i]].top()))res.push_back(tmp[i]);
        }
        return res;
    }
    // read-only で nodeID の m_NodeFactor のコピーを取得 
    // 副作用 : nodeID の m_NodeFactor を splay 
    SplayNode operator [](long long nodeID){
        m_NodeFactor[nodeID]->splay();
        SplayNode res = *(m_NodeFactor[nodeID]);
        return res.make_temp_instance();
    }
