区間 set

概要

平衡二分木で区間を管理するデータ構造です。


課題

\(N\) 個の区間 \([L_1,R_1) , [L_2,R_2) , \dots , [L_N,R_N) \) と、\(M\) 個の区間 \([S_1,T_1) , [S_2,T_2) , \dots , [S_M,T_M) \) がある。

\([L_1,R_1) , [L_2,R_2) , \dots , [L_N,R_N) \) が成す数直線上の領域と \([S_1,T_1) , [S_2,T_2) , \dots , [S_M,T_M) \) が成す数直線上の領域の共通部分の長さを答えよ。

制約

  • \(1\leq N,M \leq 2\times10^5\)
  • \( -10^9 \leq L_i < R_i \leq 10^9 \)
  • \( -10^9 \leq S_i < T_i \leq 10^9 \)


解法

前提として、区間 \([L,R)\) の長さは \(R-L\) です。

\([L_1,R_1) , [L_2,R_2) , \dots , [L_N,R_N) \) で、共有部分をもつ区間同士はマージして一つの区間にする。\([S_1,T_1) , [S_2,T_2) , \dots , [S_M,T_M) \) も同様。あとは座標圧縮して左から見ていく。


区間を平衡二分木に乗せる


以下の要件を満たすように、平衡二分木のノードに区間 \([L,R)\) を表すメンバ \(L,R\) を持たせます。

  1. \(R\) の順序でソートされている。
  2. 部分木のノードの \(R-L\) の総和を集約として持つ。
  3. 全ての区間は互いに重複しない。
    • 挿入で重複部分ができたら、それらをマージする。
  4. 隣接する区間 \([a,b) , [b,c)\) があるなら、マージして \([a,c)\) とする。

1. \(R\) でソートされている。

\(x\) をカバーするノード (区間) にアクセスすることを考えます。\(R\) が \(x\) より大きいもので、最も \(R\) が小さいノード ( 区間 \(P\) とする ) を見ることで、\(x\) 付近の区間にアクセスできます。


このとき、区間 \(P\) に \(x\) が含まれなければ、\(x\) をカバーする区間はない。逆に、\(x\) が含まれていれば \(P\) が答えです。



2. 部分木のノードの \(R-L\) の総和を集約として持つ。

\(R-L\) と言うのは、区間 \([L,R)\) の長さです。これを集約しておくことで、以下のクエリに応えることができます。

  • 管理している区間で、区間 \([a,b)\) と重複する部分の長さを答えよ。

\(a,b\) がどちらも同じ区間に含まれる場合は \(b-a\) が答えです。よって、そうでない場合を考えます。

まず、区間 \([a,b)\) に完全に覆われている部分は、\(R-L\) の集約を計算します。


\(a\) を含む区間と \([a,b)\) の重複部分の長さと、\(b\) を含む区間と \([a,b)\) の重複部分の長さを足し合わせることで答えがわかります。



3. 全ての区間は互いに重複しない。& 4. 隣接する区間はマージする。

これらの処理は、区間を追加する操作の時に行うことにします。以下は区間 \([3,7)\) を新たに追加する様子です。


まず、追加したい区間に完全にカバーされる区間は平衡二分木から削除します。


それ以外の部分 (両端) で重なる部分が存在すれば、マージして一つの区間として挿入します。この時、両端と隣接する区間もマージしておきます (要件 4)。


結果、重複部分を一つの区間にまとめて以下のようになります。


重複区間を \(1\) つにまとめるのは非直観的だと思うなら、重複部分をもつ区間の追加を禁止するように assert しましょう。実際のところ、どう処理するべきかは状況によります。

区間の削除

管理している区間から、\([a,b)\) と重なる部分だけを削除します。

この時、追加操作の時に重複していた部分も削除されることに注意してください。

区間 \([-4,-1)\) を削除

削除する部分が、既存の区間に含まれている場合、その部分を削除して既存の区間を $2$ 分割します。


区間 \([6,10)\) を削除

この場合も同様です。


区間 \([-4,11)\) を削除

削除したい区間が既存の区間に完全に含まれない場合、まず、削除したい区間に完全にカバーされている既存の区間を全て削除します。


その後、一部だけ重複している部分を縮小します。


計算量について

ある区間に注目すると、その区間は追加と削除が高々 \(1\) 度まで行われます。また、区間のマージや縮小の対象になる回数の総和はクエリの回数で抑えることができるので、クエリを通して計算量は準線形時間です。 







/*
    Copyright © (c) okoteiyu 2024. 
    Released under the MIT license(https://opensource.org/licenses/mit-license.php) 
                   
    - 複数の T 型半開区間 : [ L , R ) を管理するデータ構造(set)
    - R の値でソートして、探索するときも R の値の大小で探索
    - 重複する部分がある区間同士はマージして管理
        |- ex.[ 1 , 6 ) と [ 3 , 8 ) が同時に set に入れられた場合は [ 1 , 8 ) として管理
        |- strong_debug が true なら、上記のように、重複を持つ区間の挿入をエラーにする
    |
    - insert( l , r ) : [ l , r ) を setに追加
        |- 重複部分は 1 つにまとめる
        |- 隣接する区間は 1 つにまとめる
    - erase(l , r ) := set に入っている区間たちの中で、区間 [ l , r ) と重複している部分を削除
        |- ex. [ 1 , 6 ) , [ 9 , 11 ) が set に入っている状態で erase( 5 , 9 ) をすると、 [ 1 , 5 ) , [ 9 , 11 ) になる
        |- insert で 1 つにまとめられた部分であっても、気にせず削除する (非直観的？？)
            |- 気になるなら strong_debug を ON にする
    - T cover_length( l , r ) := set に入っている区間たちと、[ l , r ) の重複している長さを返す
        |- 区間 [ l , r ) の長さは r-l である
    |
    - int find_included( x ) := setの区間のうち、点 x を含む区間のindexを返す(存在しなければ-1を返す)
    - int find_cross( l , r ) := setの区間のうち、[ l , r ) と交差する区間のindexを+++一つ+++返す(存在しなければ-1を返す
    |
    - strong_debug が true なら、すでに存在する区間と重複する区間を insert できない
*/
template<class T , bool strong_debug = false >
class RangeSet{
    /*   
        Copyright © (c) okoteiyu 2024. 
        Released under the MIT license(https://opensource.org/licenses/mit-license.php) 
    */

    private:
    
    struct SplayNode{
        friend class RangeSet;
        public:
        // 区間は半開区間で [ L , R )
        T L,R;

        private:
        
        SplayNode *parent = nullptr;  // 親ノード
        SplayNode *left = nullptr;    // 左の子ノード
        SplayNode *right = nullptr;   // 右の子ノード

        // 部分木のうち、区間がカバーしている範囲の長さの和
        T length_sum;
        int SubTreeSize = 1;
        
        /*==================================================
            メモリアライメントに配慮して、bool は最後に宣言する
        ===================================================*/
        // ノードのコピーかどうか ( splay tree の get などに詳細 )
        bool temp_instance = false;
        
        SplayNode(){}
        SplayNode(T l , T r) : L(l) , R(r) {
            assert( L < R );
            update();
        }
        
        // デフォルトのシャローコピーを採用
        SplayNode(const SplayNode&) = default;
        SplayNode(SplayNode&&) = default;
        SplayNode& operator=(const SplayNode&) = default;
        SplayNode& operator=(SplayNode&&) = default;


        // 隣接頂点へのアクセスを禁止したコピーを返す。
        // コピーに対する splay や update などは invalid
        SplayNode make_temp_instance(){
            assert(temp_instance == false);
            SplayNode res = *this;
            res.left = nullptr;
            res.right = nullptr;
            res.parent = nullptr;
            res.temp_instance = true;
            return res;
        }
        

        // this - this->parent を回転させる (親は必ず存在するものとし、位置関係も誤りがないものとする)
        void rotate() noexcept {
            //GrandParentの子を張り替え
            if(this->parent->parent != nullptr) [[likely]] {
                if(this->parent == this->parent->parent->left)this->parent->parent->left = this;
                else this->parent->parent->right = this; 
            }
            
            //元の親に対する位置で場合分け
            if(this->parent->left == this){
                this->parent->left = this->right;
                if(this->right != nullptr)this->right->parent = this->parent;
                this->right = this->parent;
                this->parent = this->right->parent;
                this->right->parent = this;
                this->right->update();
            }else{
                this->parent->right = this->left;
                if(this->left != nullptr)this->left->parent = this->parent;
                this->left = this->parent;
                this->parent = this->left->parent;
                this->left->parent = this;
                this->left->update();
            }

            this->update();
            return;
        }


        /*
            自分から見た親の位置
            0 -> 親頂点がない
            1 -> 親の左の場合
            2 -> 親の右の場合
        */
        int state() noexcept {
            if(this->parent == nullptr) [[unlikely]] return 0;
            if(this->parent->left == this)return 1;
            return 2;
        }
        
        // あるNodeを回転を駆使し一番上まで持っていく
        void splay(){
            while(this->parent != nullptr){
                if(this->parent->state() == 0){
                    this->rotate();
                    break;
                }
                if( this->parent->state() == this->state() )this->parent->rotate();
                else this->rotate();
                this->rotate();
            }
            this->update();// while が呼ばれなかった場合のため
            return;
        }

 
        // サイズなど、Nodeの持っている情報をupdateする(順番大事)
        void update(){
            assert(temp_instance == false);
            
            this->SubTreeSize = 1;
            this->length_sum = abs(this->R - this->L);
            if(this->left != nullptr){
                this->SubTreeSize += this->left->SubTreeSize;
                this->length_sum += this->left->length_sum;
            }
            if(this->right != nullptr){
                this->SubTreeSize += this->right->SubTreeSize;
                this->length_sum += this->right->length_sum;
            }
            return;
        }

        // ノードの値によって順序を定義。ただし、共に遅延評価がなく、正しい情報を持っている必要がある。
        // {Key,Value} の辞書順
        friend bool operator<(const SplayNode& a , const SplayNode& b) noexcept {
            return bool(a.R < b.R);
        }

    };
 
    protected:
    /*=================================
                メンバ変数たち
    =================================*/
                
    
    // 管理する木の根ノード
    SplayNode *m_Root = nullptr;
    // upper_bound などで比較を行うためのノードオブジェクト。SplayTree に挿入はしない
    SplayNode* const m_bluff_object = new SplayNode();

    private:   
 
    // rootを根とする部分木において、求めたいindexまで降りていく(左側の部分木サイズを参照する)
    static SplayNode *get_sub(int index, SplayNode* root){
        if(root == nullptr)return root;
        assert(root->parent == nullptr);

        auto now = root;
        while(true){
            int left_size = bool(now->left == nullptr) ? 0 : now->left->SubTreeSize;
            if(index == left_size)break;
            if(index < left_size){
                now = now->left;
            }else{
                now = now->right;
                index -= left_size+1;
            }
        }
        now->splay();
        return now;
    }

 

    /*==============================================================================
        merge/split ベースの実装を行う。
        |- merge(lef,rig) : rig の左の子に lef をつける
        |- split(i,rt) : i 番目のノードの左の辺を無くし、(lef,rig) のペアを返す
        merge と split が反対の操作であることを利用して計算量を削減する
    ==============================================================================*/

    
    // leftroot を根とする SplayTree と、rightroot を根とする SplayTree をこの順に結合する。
    static SplayNode *merge(SplayNode *leftRoot, SplayNode *rightRoot){
        if(leftRoot == nullptr)return rightRoot;// 右のみ
        if(rightRoot == nullptr)return leftRoot;// 左のみ
        assert(leftRoot->parent == nullptr && rightRoot->parent == nullptr);
        // 怖いのでここで update かけときます。大したコストではないです 
        leftRoot->update();
        rightRoot->update();

        rightRoot = get_sub(0,rightRoot);
        rightRoot->left = leftRoot;
        leftRoot->parent = rightRoot;
        rightRoot->update();
        return rightRoot;
    }
    
    // root が根の SplayTree を、左から i-1 番目と i 番目の間で分割する。
    std::pair<SplayNode*,SplayNode*> split(int i, SplayNode*root){
        if(root == nullptr)return std::make_pair(nullptr , nullptr);
        assert(root->parent == nullptr);
        assert(0 <= i && i <= root->SubTreeSize);
        if(i == 0)return std::make_pair(nullptr , root);
        if(i == root->SubTreeSize)return std::make_pair(root, nullptr);
        root = get_sub(i , root);
        auto leftRoot = root->left;
        auto rightRoot = root;
        if(rightRoot != nullptr)rightRoot->left = nullptr;
        if(leftRoot != nullptr)leftRoot->parent = nullptr;
        leftRoot->update();
        rightRoot->update();
        return std::make_pair(leftRoot,rightRoot);
    }
    

    /*
        root が根であるような Splay 木で、以下の条件を満たすノード x の個数を返す。
        lower = true : x < node (lower_bound)
        lower = false: x <= node (upper_bound)
        条件を満たすギリギリの境界を探索する (ただし、境界の左右どちらかは不定)
    */
    static std::pair<SplayNode*,int> bound_sub(SplayNode* Node, SplayNode *root , bool lower){
        if(root == nullptr)return std::make_pair(root,0);
        SplayNode *now = root;// 条件による境界ギリギリのノードを根から探索する
        Node->update();// 念の為集約と遅延評価をしておく
        bool satisfy = false;
        while(true){
            if(lower)satisfy = bool(*now < *Node); // lower_bound (now < Node)
            else satisfy = !bool(*Node < *now);    // upper_bound (now <= Node)
            // now が条件を満たす側にいる場合、右側に境界があるとわかる
            if(satisfy){
                if(now->right == nullptr)break;
                now = now->right;
            }else{
                if(now->left == nullptr)break;
                now = now->left;
            }
        }
        int res = 0;
        // now が条件を満たす側で終了した場合、自身の分を加算する。
        if(satisfy)res = 1;
        now->splay();
        if(now->left != nullptr)res += now->left->SubTreeSize;
        return std::make_pair(now ,res);
    }


    // root が根の splaytree に NODE (別の連結成分)を挿入する
    // NODE は update されている必要がある
    SplayNode *insert_sub(SplayNode *NODE , SplayNode *root){
        NODE->update();// 念の為集約と遅延評価をしておく
        if(root == nullptr)return NODE;
        
        // lower_bound で、x < NODE の境界となる x を取得 (ただし境界の右側か左側かは不定)
        root = bound_sub(NODE,root,true).first;
        // root が x < NODE の境界の左右どちらかによって、NODE を配置する場所が変わる
        if(*root < *NODE){ // root < NODE の場合は root の右に
            if(root->right != nullptr)root->right->parent = NODE;
            NODE->right = root->right;
            root->right = nullptr;
            NODE->left = root;
        }else{// NODE <= root の場合は root の左に
            if(root->left != nullptr)root->left->parent = NODE;
            NODE->left = root->left;
            root->left = nullptr;
            NODE->right = root;
        }
        root->parent = NODE;
        root->update();
        NODE->update();
        return NODE;// NODE が根になっている
    }
     


    // upper_bound などで比較を行うためのノードオブジェクト。SplayTree に挿入はしない
    SplayNode* const BluffObject(T l , T r){
        m_bluff_object->L = l;
        m_bluff_object->R = r;
        return m_bluff_object;
    }


    public:
    RangeSet(){}
    ~RangeSet(){
        // メモリ解放。消すと早くなる。
        while(m_Root != nullptr)this->pop(0);
    }
        // コピー
    RangeSet(const RangeSet<T, strong_debug>& param_tree) : RangeSet() {
        // subr 以降の部分木をコピー
        function<SplayNode*(const SplayNode*)>subr_copy = [&](const SplayNode* subr){
            if(subr == nullptr)return static_cast<SplayNode*>(nullptr);
            SplayNode* res = new SplayNode(*subr);
            res->left = subr_copy(res->left);
            res->right = subr_copy(res->right);
            if(res->left != nullptr)res->left->parent = res;
            if(res->right != nullptr)res->right->parent = res;
            return res;
        };
        m_Root = subr_copy(param_tree.m_Root);
    }
    
    // ムーブ
    RangeSet(RangeSet<T, strong_debug>&& param_tree) noexcept : RangeSet() {
        swap(this->m_Root , param_tree.m_Root);
    }

    // コピー&スワップ イディオム
    RangeSet& operator=(RangeSet<T, strong_debug> param_tree){
        swap(this->m_Root , param_tree.m_Root);
        // param_tree に渡したリソースは param_tree のデストラクタで破棄される
        return *this;
    }
 
    // 格納されている (連続していない) 区間数
    int size(){
        if(m_Root==nullptr)return 0;
        return m_Root->SubTreeSize;
    }
    // 左から i 番目のNodeのコピーを取得 (0-index)
    // ただし、値の取得が目的なので right などの隣接頂点へのアクセスを封印する
    SplayNode get(int i){
        assert(0 <= i && i < size());
        m_Root = get_sub(i,m_Root);
        return m_Root->make_temp_instance();
    }


    // [l,r) に対応する部分木の根のコピーを返す。ただし、隣接ノードへのアクセスを封じる。
    // 使用用途はモノイド積の取得に限定 (コピーなので、オリジナルの現在の値とは関係なく [l,r) のデータが格納されている)
    SplayNode GetRange(int l , int r){
        assert(0 <= l && l < r && r <= size());
        std::pair<SplayNode*,SplayNode*> tmp = split(r,m_Root);
        SplayNode* rightRoot = tmp.second;
        tmp = split(l,tmp.first);// 部分木を取り出す。
        SplayNode res = tmp.second->make_temp_instance();
        m_Root = merge(merge(tmp.first,tmp.second),rightRoot);
        return res;
    }

    // index番目のNodeを消去
    void pop(int index){
        assert(0 <= index && index < size());
        SplayNode *center = get_sub(index,m_Root);
        SplayNode *leftRoot = center->left;
        SplayNode *rightRoot = center->right;
        if(leftRoot != nullptr)leftRoot->parent = nullptr;
        if(rightRoot != nullptr)rightRoot->parent = nullptr;
        center->left = nullptr;
        center->right = nullptr;
        center->update();

        m_Root = merge(leftRoot,rightRoot);
        delete center;
    }
    
    // R が x 未満の頂点の個数を返す
    int lower_bound(T x){
        if(size() == 0)return 0;
        std::pair<SplayNode*,int> tmp = bound_sub(BluffObject(x,x),m_Root,true);
        m_Root = tmp.first;
        return tmp.second;
    }
    
    // Rがx以下の頂点の個数を返す
    int upper_bound(T x){
        if(size() == 0)return 0;
        std::pair<SplayNode*,int> tmp = bound_sub(BluffObject(x,x),m_Root,false);
        m_Root = tmp.first;
        return tmp.second;
    }

    // 点 x をカバーする区間のindex(存在しなければ-1を返す)
    int find_included(T x){
        int it_ = upper_bound(x);
        if(it_ < 0 || it_ >= size())return -1;
        if(get(it_).L <= x)return it_;
        return -1;
    }

    // 二つの区間 [ l1 , r1 ) と [ l2 , r2 ) が 交差するか
    inline static bool intersect(T l1 , T r1 , T l2 , T r2){
        return (l1 <= l2 && l2 < r1) || (l2 <= l1 && l1 < r2);
    }

    // 二つの区間 [ l1 , r1 ) と [ l2 , r2 ) の共通部分の長さ
    inline static T CommonArea(T l1 , T r1 , T l2 , T r2){
        if(!intersect(l1,r1,l2,r2))return 0;
        return min(r1,r2)-max(l1,l2);
    }
    
    
    // 区間 [ lef , rig ) と交差する区間を +++ 一つ +++ 見つける(indexで返す)
    int find_cross(T lef , T rig){
        assert(lef < rig);
        int it_ = upper_bound(lef);// 右端が l よりも真に大きい区間
        if(it_ >= 0 && it_ < size()){
            SplayNode x = get(it_);
            if(intersect(lef,rig,x.L,x.R))return it_;// 被った区間のindexを返す   
        }
        return -1;// なければ-1を返す
    }
    
 
    // 区間 [ lef , rig ) を追加
    void insert( T lef , T rig){
        assert(lef < rig);
        SplayNode x;
        while(true){// 被っている区間を全てsetから取り出して、[ l ,r ) と一つにmergeしてからsetに入れ直す
            int it_ = find_cross(lef,rig);
            if(it_ < 0 || it_ >= size())break;

            if(strong_debug == true) throw(std::runtime_error("重複する区間の挿入は禁止されています。"));
            
            x = get(it_);
            // 区間をマージ
            lef = min(lef,x.L);
            rig = max(rig,x.R);
            this->pop(it_);// 重複する区間を削除
        }
        // 隣接している区間をマージ
        int it_ = upper_bound(lef);
        // ループ内でDeleteするので、大きいindexから見る
        for(int itr = it_ ; itr >= it_ - 1 ; itr-- ){
            if(itr >= 0 && itr < size()){
                x = get(itr);
                // 以下のif条件はどちらも独立
                if(x.R == lef){
                    lef = min(lef,x.L);
                    this->pop(itr);
                }else if(x.L == rig){
                    rig = max(rig,x.R);
                    this->pop(itr);
                }
            }
        }
        m_Root = insert_sub(new SplayNode(lef , rig) ,m_Root);
        return;
    }
 
    
    // 格納されている区間に対して、lef,rig) と重複する部分を全て削除する。
    void erase(T lef , T rig){
        assert(lef<rig);
        SplayNode x;
        while(true){//被っている区間を見つけて、被っている部分を消していく
            int it_ = find_cross(lef,rig);
            if(it_ < 0 || it_ >= size())break;
            x = get(it_);
            this->pop(it_);
            if(lef <= x.L && x.R <= rig)continue;//すっぽりと削除区間に含まれる場合
            else if(x.L <= lef && rig <= x.R){//削除区間がすっぽりと含まれている場合(穴が空いたような状態に)
                insert(x.L,lef);
                insert(rig,x.R);
            }else if(lef <= x.L && x.L < rig)insert(rig,x.R);//左側が削除区間に含まれている場合    
            else if(lef < x.R && x.R <= rig)insert(x.L,lef);//右側が削除区間に含まれている場合
        }
        return;
    }
    
    // setに入っている区間で、区間 [ lef , rig ) と被っている部分の長さの総和
    // ただし、区間 [ lef , rig ) の長さは rig-lef であるとする
    T cover_length(T lef , T rig){
        assert(lef < rig);
        int it_lef = upper_bound(lef);
        if(it_lef == size())return 0;
        int it_rig = upper_bound(rig);
        T res = 0;//答え
        SplayNode x = get(it_lef);
        if(x.L <= lef && rig <= x.R)return rig-lef;
        if(x.L<lef){
            it_lef++;
            res += x.R - lef;
        }
        if(it_lef < it_rig)res += GetRange(it_lef,it_rig).length_sum;
        if(it_rig < size()){
            x = get(it_rig);
            if(x.L < rig)res += rig-x.L;
        }
        return res;
    }
    
    void Debug(){
        cerr << "DEBUG" << endl;
        for(int i = 0 ; i < size();i++)cerr << "[ " <<  get(i).L << " , " << get(i).R << " )" << endl;
    }
    // read-only で、i 番目の区間にアクセス
    SplayNode operator [](int index){
        return get(index);
    }
