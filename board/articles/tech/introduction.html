てゆ掲示板について
初めまして、おこていゆと申します。この度、okoteiyu.com のコンテンツを拡張して、「てゆ掲示板」なるものを作成しました。
記事のデザインは 2026/5/8 の時点では仮のものになっていますが、今後も改修していく予定です。
ここでは、ページの使い方を簡単に説明します。まず特定の記事を共有したい場合、てゆ掲示板自体のリンクに、URL パラメータとして「?post_id=[記事id]」を追加します。記事 id は、掲示板のポストに割り振られている固有の ID です。この記事の場合は T0000 になります。
記事 ID 付きで共有された掲示板ページをリロードする時は、上部バーの「No Param」を押してください。そうすることで、URL パラメータを消した状態に戻すことができます。
他には？
ポストの「カテゴリ」の左隣に「🔗」文字があるものは、タップすることで詳細な記事内容を展開できます。ないものはポスト単体のものです。
