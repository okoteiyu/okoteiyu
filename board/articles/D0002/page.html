WEB 開発の経緯

あれは 2022 年ごろのこと。当時、競技プログラミングの成績で AtCoder レーティング 1200 を達成した後、しばらく成績が停滞していた時期でした。その頃、競技プログラミングの成果として AtCoder レーティング以外の指標を求めていたオレは、自分の C++ ライブラリを作ると同時に、アルゴリズムとデータ構造の知識を記事にしてまとめようと考えたわけダ。その頃の web サイトが以下だよ。
ご覧の通り酷い有様でし。見た目もそうだし、サイトの設計自体もとても見られたものではないんだよね💦 全てのページがリンク同士で繋がってる構造だったし、ページ間で一貫したルールがあったりもしなかったし...
その後、Doc-View スタイルを真似した index ページに変更しましたけど、項目ごとに愚直な直書き HTML で全て表現するというバッドプラクティスでした。
その後は AtCoder 水色レベルの知識をまとめつつ、記事も少しずつ作成しておりました。転機が訪れたのは AtCoder レーティングが 1600 に到達した後、当時のサイトの諸々の不便さを鑑みて、サイト全体を改装するに至りました。
今回の改装では、ちゃんと Doc-View 形式を採用しつつ、HTML 文自体もなるべく効率的なデザインにしました!!! これで index ページの基本形は概ね現在の形になりました。
index ページの形はできましたが、この頃は肝心の記事の中身の様式が定まっておりませんでした。この後しばらくは記事の様式や、開発ルール/Wiki の整備があり、最終的にデザインも含めて現在の学習記事公開ページになりました。
当初はアルゴリズムとデータ構造のことしか投稿していませんでしたが、その後は他の分野の記事も投稿するようになりました。
そして今、学習記事だけではなく、okoteiyu.com はさまざまなコンテンツを載せるサイトにしようとしているのです。これからの進化に期待していてください!!!
